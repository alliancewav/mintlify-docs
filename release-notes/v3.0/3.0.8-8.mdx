## Summary

A major update to the Producer Dashboard, bringing visual consistency with the player interface, a new comprehensive finances section, and significant performance optimizations.

## What's New

- **Unified Design**: The Producer Dashboard now shares the same layout, navigation, and visual style as the main web player for a seamless experience.
- **Finances Dashboard**: A new, centralized hub for tracking earnings, viewing transaction history, and managing Stripe Connect settings.
- **Performance Boost**: Optimized API architecture reduces network requests by 50% on key dashboard pages.
- **Real-Time Data**: Fixed caching issues to ensure XP and achievement progress updates appear within 1 minute.

## Why It Matters

- **Better Workflow**: Moving between the player and dashboard is now fluid and intuitive.
- **Financial Clarity**: You can now see exactly where your revenue comes from with detailed transaction breakdowns.
- **Faster Loading**: Pages load significantly faster due to reduced API calls and optimized caching.

## How to Use

- **Finances**: Navigate to `Dashboard > Finances` to view your new earnings overview.
- **Navigation**: Use the new sidebar (matching the player) to access all producer tools.
