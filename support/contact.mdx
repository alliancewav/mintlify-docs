## Escalation

If your issue isn't resolved satisfactorily:

1. Reply to the support thread with additional context
2. Request escalation in your response
3. Allow additional time for senior review

### When to Escalate

| Issue Type | When to Escalate |
|------------|------------------|
| Standard support | After 48 hours without resolution |
| Payment/payout issues | After 5 business days |
| Account suspension | If appeal not acknowledged in 3 days |
| Security concerns | Immediately to security@beatpass.ca |

### Escalation Information to Include

For faster escalation, provide:

| For Payout Issues | For Licensing Issues |
|-------------------|---------------------|
| Stripe account email | License UUID |
| Transaction dates | Purchase date |
| Expected vs received amounts | Track name |
| Previous ticket numbers | Buyer/seller info |

### Critical vs Standard Issues

| Critical (24hr target) | Standard (48hr target) |
|------------------------|------------------------|
| Can't access account | Feature not working |
| Payment charged incorrectly | Upload questions |
| Security breach suspected | General questions |
| Active fraud | Feedback |
