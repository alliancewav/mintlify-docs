## Overview

BeatPass is fully accessible on mobile devices through our responsive web application. This guide covers the mobile experience and tips for optimal use.

---

## Accessing on Mobile

### Mobile Web App

BeatPass works in any modern mobile browser:

1. Open your mobile browser (Chrome, Safari, Firefox)
2. Navigate to `beta.beatpass.ca`
3. Log in or create an account
4. Start browsing and listening

### Add to Home Screen

For an app-like experience:

<Tabs>
  <Tab title="iOS (Safari)">
    1. Open BeatPass in Safari
    2. Tap the Share button (box with arrow)
    3. Scroll and tap "Add to Home Screen"
    4. Name it and tap "Add"
  </Tab>
  <Tab title="Android (Chrome)">
    1. Open BeatPass in Chrome
    2. Tap the menu (three dots)
    3. Select "Add to Home Screen"
    4. Confirm the name and tap "Add"
  </Tab>
</Tabs>

<Tip>
  Adding to home screen enables fullscreen mode and faster loading.
</Tip>

## Mobile Features

### What Works on Mobile

| Feature              | Mobile Support |
| -------------------- | -------------- |
| Browsing & Discovery | Full           |
| Streaming & Playback | Full           |
| Search & Filters     | Full           |
| Saving & Library     | Full           |
| Profile & Settings   | Full           |
| Messaging            | Full           |
| Purchasing Licenses  | Full           |
| Basic Uploading      | Limited        |
| Analytics Dashboard  | Optimized      |

### Mobile Player

The mobile player includes:

- Play/pause controls
- Skip forward/back
- Queue management
- Volume control
- Track information
- Like and save buttons

### Gesture Controls

| Gesture            | Action             |
| ------------------ | ------------------ |
| Tap waveform       | Seek to position   |
| Swipe up on player | Expand full player |
| Swipe down         | Minimize player    |
| Double tap         | Like track         |

## Mobile Optimization

### Data Usage

Control data consumption:

| Setting      | Data Usage        |
| ------------ | ----------------- |
| High Quality | ~2MB per minute   |
| Normal       | ~1MB per minute   |
| Data Saver   | ~0.5MB per minute |

Configure at: Settings \> Playback \> Audio Quality

### Offline Listening

<Note>
  Offline mode is planned for future updates. Currently, streaming requires an internet connection.
</Note>

### Battery Optimization

- Close unused browser tabs
- Use dark mode in low-light conditions
- Disable autoplay when not actively listening
- Lower screen brightness

## Supported Devices

### Minimum Requirements

| Platform | Requirement                 |
| -------- | --------------------------- |
| iOS      | iOS 14+ with Safari         |
| Android  | Android 8+ with Chrome      |
| Tablets  | Same as mobile requirements |

### Recommended

- Latest iOS or Android
- Modern browser updated to latest version
- Stable internet connection
- At least 2GB RAM

### Known Limitations

| Device/Browser    | Limitation                 |
| ----------------- | -------------------------- |
| Older Android     | May have playback issues   |
| In-app browsers   | Some features may not work |
| Private/Incognito | Session won't persist      |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Audio stops when screen locks">
    **iOS**: Audio should continue. If not, check Focus/Do Not Disturb settings. **Android**: Ensure BeatPass isn't restricted by battery optimization.
  </Accordion>
  <Accordion title="Page not loading correctly">
    - Clear browser cache
    - Try closing and reopening browser
    - Check internet connection
    - Try a different browser
  </Accordion>
  <Accordion title="Login issues">
    - Clear cookies for beatpass.ca
    - Disable any VPN or proxy
    - Try "Forgot Password" if needed
  </Accordion>
  <Accordion title="Player not appearing">
    - Refresh the page
    - Check if audio is muted system-wide
    - Try force-closing and reopening browser
  </Accordion>
</AccordionGroup>

## Mobile vs Desktop

### When to Use Mobile

- Quick listening sessions
- Checking notifications
- Responding to messages
- Monitoring analytics
- Browsing new releases

### When to Use Desktop

- Uploading new beats
- Detailed analytics review
- Bulk metadata editing
- Complex searches/filtering
- License purchases (for better review)
