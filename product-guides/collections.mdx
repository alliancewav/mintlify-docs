## Overview

Collections are curated groups of tracks that help organize and showcase content on BeatPass. They differ from playlists in that they're designed for discovery and curation rather than personal listening queues.

<Info>
  **Audience:** All users (creation requires authentication)
</Info>

---

## Types of Collections

### User Collections

Collections you create to organize tracks:

| Feature | Description |
|---------|-------------|
| **Public** | Visible to all users, appears in search |
| **Private** | Only visible to you |

### Collection Limits

| Limit | Value |
|-------|-------|
| Max collections per user | 50 |
| Max tracks per collection | 200 |
| Name length | 100 characters |
| Description length | 500 characters |

### Featured Collections

Platform-curated collections showcasing:

- New releases
- Genre spotlights
- Trending tracks
- Seasonal content

<Note>
  Featured collections are curated by BeatPass editors and cannot be created by users.
</Note>

### How Featured/Recommended Are Chosen

| Type | Selection Method |
|------|------------------|
| **Featured** | Editorial curation by BeatPass team based on quality, relevance, and timeliness |
| **Recommended** | Algorithmic suggestions based on your listening history, followed artists, and genre preferences |

Featured collections appear on the homepage and discovery pages. Recommendations are personalized and shown in your feed.

## Creating a Collection

<Steps>
  <Step title="Navigate to Collections">
    Go to your profile or the Collections section.
  </Step>
  
  <Step title="Click Create Collection">
    Start a new collection.
  </Step>
  
  <Step title="Add Details">
    | Field | Description |
    |-------|-------------|
    | Name | Collection title |
    | Description | What the collection is about |
    | Cover Image | Optional custom artwork |
    | Visibility | Public or private |
  </Step>
  
  <Step title="Add Tracks">
    Browse or search for tracks to add.
  </Step>
  
  <Step title="Save">
    Your collection is now available.
  </Step>
</Steps>

## Managing Collections

### Adding Tracks

Add tracks to your collections from:

- Track page (click "Add to Collection")
- Search results
- Artist pages
- Within the collection editor

### Reordering Tracks

Drag and drop tracks within a collection to change their order.

### Editing Collections

Update collection details anytime:

1. Open the collection
2. Click **Edit**
3. Modify name, description, or visibility
4. Save changes

### Deleting Collections

<Warning>
  Deleting a collection is permanent. The tracks themselves are not affected.
</Warning>

1. Open the collection
2. Click **Delete**
3. Confirm deletion

## Collections vs Playlists

| Feature | Collections | Playlists |
|---------|-------------|-----------|
| **Purpose** | Curation & discovery | Personal listening |
| **Visibility** | Public/private | Public/private |
| **Featured** | Can be featured | Not featured |
| **Playback** | Browse-focused | Queue-focused |
| **Collaboration** | Single owner | Can invite editors |

---

## API Access

Collections are accessible via the API for authenticated users.

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/collections` | List collections |
| GET | `/collections/featured` | Featured collections |
| GET | `/collections/recommendations` | Personalized suggestions |
| POST | `/collections` | Create collection |
| PUT | `/collections/{id}` | Update collection |
| DELETE | `/collections/{id}` | Delete collection |

For full API documentation, see [API Endpoints - Collections](/api/endpoints#collections).
