# Profile Elements

Your producer profile combines branding, identity, and discovery metadata. This page covers the profile elements you can edit today.

## Cover Image

Your cover image is the large banner at the top of the producer page.

### Current File Requirements

- **Formats**: JPEG, JPG, PNG, GIF, or WebP
- **Max size**: 2MB

### How to Update It

<Steps>
  <Step title="Open your producer profile">
    Navigate to your producer profile page on BeatPass.
  </Step>
  <Step title="Open the editor">
    Click the **More** button on your profile, then select **Edit**.
  </Step>
  <Step title="Update the image fields">
    Use the producer editor to replace the cover image and related profile artwork.
  </Step>
  <Step title="Save the profile">
    Save once the preview looks right.
  </Step>
</Steps>

<Tip>
Use artwork that stays readable behind your producer name and action buttons.
</Tip>

## Producer Avatar

Your avatar appears in the header, around your catalog, and in other producer surfaces.

### Current File Requirements

- **Formats**: JPEG, JPG, PNG, GIF, or WebP
- **Max size**: 2MB

Use the same producer editor entry point to replace the avatar.

<Tip>
Square images work best because the public page renders the avatar in a circular frame.
</Tip>

## Location

Location is part of your public producer biography.

### Where to Edit It

Open **Biography** in the producer editor to change:

- **City**
- **Country**

## Genres

Genres help BeatPass present your sound in discovery and profile contexts.

### Where to Edit Them

1. Open your producer editor
2. Go to **Details**
3. Update the **Genres** field
4. Save the profile

## Summary

| Element | Where to manage it |
| --- | --- |
| **Cover image** | Producer editor |
| **Avatar** | Producer editor |
| **City and country** | **Biography** tab |
| **Genres** | **Details** tab |

<Note>
The public profile header does not use a separate inline "change cover" control. Use **More -> Edit** when you need to change profile artwork or metadata.
</Note>
