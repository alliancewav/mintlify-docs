# Discography & Profile Tabs

Your producer profile uses a fixed set of public tabs, with a few tabs appearing only when the related data or permissions are available.

## Available Tabs

### Discography

The **Discography** tab is the main producer showcase. It can include:

- Your pinned track
- Popular beats
- New beats
- Albums
- About information
- Professional credits

### Similar Artists

The **Similar artists** tab appears when BeatPass has matching related-profile data for your page.

### Gallery

The **Gallery** tab is for profile images and photos. BeatPass can keep this tab available even before you upload images, using an empty state until you add content.

### Beats

The **Beats** tab is the clearest way for visitors to browse just your tracks.

### Albums

The **Albums** tab is the collection view for your producer profile.

### Followers

The **Followers** tab shows the accounts following that producer profile.

### Achievements

The **Achievements** tab is only shown to the owner of the profile or admins.

## Tab Visibility

| Tab | When it appears |
| --- | --- |
| **Discography** | Always |
| **Similar artists** | When BeatPass has related-profile data |
| **Gallery** | When the gallery feature is enabled |
| **Beats** | Always |
| **Albums** | Always |
| **Followers** | Always |
| **Achievements** | Owner or admin only |

## Important Distinction

Professional Credits are part of the **Discography** experience. They are not a separate top-level producer tab in the current profile layout.
