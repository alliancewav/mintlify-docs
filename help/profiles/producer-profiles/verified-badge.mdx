# Verified Badge <Icon icon="circle-check" color="#449eff" />

The verified badge marks a producer profile that BeatPass has confirmed as authentic.

## Where It Appears

When a producer profile is verified, the badge appears:

- Next to your producer name on the producer profile
- On the linked user profile for that verified producer, when applicable

## What Verification Means

The badge indicates:

- **Authentic identity** — BeatPass has reviewed the profile as the real producer or brand
- **Account trust** — Visitors can distinguish the page from impersonation or confusion

<Note>
Verification is about authenticity, not popularity. It is not a ranking badge.
</Note>

## How to Request Verification

BeatPass currently routes verification requests through the Producer Program flow.

<Steps>
  <Step title="Open the Producer Program request flow">
    Start from the Producer Program page.
  </Step>
  <Step title="Choose Get verified">
    Select **Get verified** from the available request types.
  </Step>
  <Step title="Complete the request">
    Fill in the information BeatPass asks for.
  </Step>
  <Step title="Follow the verification instructions">
    The current request flow references BeatPass's Discord community as part of the verification path.
  </Step>
  <Step title="Wait for review">
    If BeatPass approves the request, the badge appears automatically on the producer profile.
  </Step>
</Steps>

<Tip>
A complete producer profile makes manual review easier. Add your bio, links, imagery, and public catalog before requesting verification.
</Tip>

## Frequently Asked Questions

### Can I lose my verified badge?

Yes. BeatPass can revoke verification if the account no longer meets platform requirements or authenticity standards.

### Does verification cost anything?

No. Verification is not a paid upgrade.

## Verification vs Subscription

Verification and paid plans are different things:

| Feature | Verification | Subscription |
| --- | --- | --- |
| **Purpose** | Identity confirmation | Paid platform benefits |
| **How you get it** | Producer Program review | Billing signup |
| **Visual result** | Blue verified badge | Subscription status on the account |

<Note>
You do not need a paid plan to request verification, and verification does not replace any subscription feature.
</Note>
