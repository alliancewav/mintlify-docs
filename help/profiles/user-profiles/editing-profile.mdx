# Editing Your Profile

Use the profile editor to update the information attached to your BeatPass account.

## Opening the Profile Editor

<Steps>
  <Step title="Open your profile page">
    Use the avatar menu and choose **Profile page**, or navigate to your user profile directly.
  </Step>
  <Step title="Click Edit">
    On your profile page, click the **Edit** button. This opens the profile editor dialog.
  </Step>
</Steps>

## What You Can Edit

| Field | Description |
| --- | --- |
| **Profile Picture** | Your public avatar |
| **Username** | Your public handle and profile URL slug |
| **First Name** | Optional first name |
| **Last Name** | Optional last name |
| **City** | Optional city |
| **Country** | Optional country |
| **Description** | Bio text |
| **Links** | External profile and website links |

<Note>
Not all fields you can edit are visibly displayed on your public profile page today. Your avatar, username, and core account identity are always shown. Other fields like description, city, country, and links are stored on your account but may not appear on the live user profile page.
</Note>

## What This Editor Is For

This editor controls your **user profile** details. Use it for:

- Avatar and username changes
- Name, location, and bio updates
- External links

It is **not** the place for:

- Email or password changes (use Account Settings)
- Billing changes (use the Billing page)
- Producer profile branding and catalog edits (use **More** → **Edit** on your producer profile)

## Saving Your Changes

1. Review your changes in the editor
2. Click **Save** to apply
3. The dialog closes and your profile updates immediately

Click **Cancel** to discard any unsaved changes.

## Avatar Tips

- **Use a square image** — Your photo displays in a circle
- **Keep it clear** — Choose an image where your face or logo is easily visible
- **Stay appropriate** — Your avatar appears across BeatPass in comments, messages, and more

<Warning>
Your profile picture is visible to everyone on BeatPass. Make sure it follows community guidelines.
</Warning>

## For Producers

If you also have a producer profile, your user profile can show:

- A **Producer Profile** button linking to your artist page
- A **verified badge** if your producer profile is verified

To edit the producer profile itself, go to the producer page and use **More** → **Edit**.

<Note>
Your user profile and producer profile are separate but connected. Edit each one from its own page.
</Note>
