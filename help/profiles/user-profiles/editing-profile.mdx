# Editing Your Profile

Use the user-profile editor when you want to change what appears on your public account page.

## Opening the Profile Editor

<Steps>
  <Step title="Open your profile page">
    Use the avatar menu and choose **Profile page**, or open your user profile directly.
  </Step>
  <Step title="Click Edit">
    On your profile page, click the **Edit** button. This opens the profile editor dialog.
  </Step>
</Steps>

## What You Can Edit

| Field | Description |
| --- | --- |
| **Profile Picture** | Your public avatar |
| **Username** | Your public handle and profile URL slug |
| **First Name** | Optional first name |
| **Last Name** | Optional last name |
| **City** | Optional city |
| **Country** | Optional country |
| **Description** | Public bio text |
| **Links** | Public external profile and website links |

<Tip>
Only add information you want visible on your public profile. Blank fields stay off the page.
</Tip>

## What This Editor Is For

This editor controls your **public user profile**. It is the right place for:

- Username
- Bio
- Profile location
- External links

It is **not** the place for:

- Email changes
- Password changes
- Billing changes
- Producer profile branding and catalog edits

## Saving Your Changes

1. Review your changes in the editor
2. Click **Save** to apply your updates
3. The dialog will close and your profile will be updated immediately

If you change your mind, click **Cancel** to discard your changes.

## Avatar Tips

- **Use a square image** — Your photo will be displayed in a circle
- **Keep it clear** — Choose an image where your face or logo is easily visible
- **Stay professional** — Your avatar appears across BeatPass in comments, messages, and more

<Warning>
Your profile picture is visible to everyone on BeatPass. Make sure it follows the community guidelines and doesn't contain inappropriate content.
</Warning>

## For Producers: Connecting to Your Producer Profile

If you have a producer profile on BeatPass, visitors to your user profile will see:

- **Producer Profile button** — A button that links directly to your producer profile where your beats are displayed
- **Verified badge** — If your producer profile is verified, the blue checkmark appears on your user profile too

To edit the producer profile itself, go to the producer page and use **More -> Edit**.

<Note>
Your user profile and producer profile are separate but connected. Edit each one from its respective profile page for the best experience.
</Note>
