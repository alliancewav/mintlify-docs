# User Profiles

Your user profile is the public page attached to your BeatPass account. It shows your identity, your public activity tabs, and any linked producer presence.

## What Visitors See

Your profile always displays:

- Your **avatar**
- Your **username**
- A **verified badge** when your linked producer profile is verified

### Profile Header

Header actions depend on who is viewing:

| Viewer | Actions |
| --- | --- |
| **You** | **Edit** |
| **Other users** | **Follow**, **Message** (when messaging is available) |
| **Linked to a producer page** | A **Producer Profile** button also appears |

<Note>
The profile editor lets you save additional details like first name, last name, city, country, description, and links. However, not all of these fields are visibly displayed on the live user profile page today. Your avatar and username are always shown.
</Note>

## Profile Tabs

Your profile has several tabs that display different types of content:

| Tab | What it shows |
| --- | --- |
| **Liked Beats** | Tracks you liked |
| **Public Playlists** | Playlists you marked public |
| **Reposts** | Reposted content, when reposts are enabled |
| **Liked Collections** | Collections you liked |
| **Liked Producers** | Producer profiles you liked |
| **Followers** | People following your user profile |
| **Following** | User accounts you follow |

## Linked Producer Profile

If you are also a producer:

- Your user profile can show a **Producer Profile** button linking to your artist page
- A verified producer badge also appears on your user profile
- Your producer catalog lives on the producer page, not the user page

<Note>
User profiles and producer profiles are related, but they are edited in different places.
</Note>
