# User Profiles

Your user profile is the public page attached to your BeatPass account. It shows your identity, your public listening activity, and any linked producer presence.

## What visitors see

Your profile can show:

- Your avatar
- Your username
- Your first and last name, if set
- Your bio, city, country, and links, if set
- A verified badge when your linked producer profile is verified
- Your public profile tabs

### Profile Header

Header actions depend on who is viewing the page:

- On **your own** profile, you see **Edit**
- On **someone else's** profile, BeatPass can show **Follow**
- The **Message** button appears when messaging is available for that view
- If the account is linked to a producer page, BeatPass can show **Producer Profile**

## Profile Tabs

Your profile has several tabs that display different types of content:

| Tab | What it shows |
| --- | --- |
| **Liked Beats** | Tracks you liked |
| **Public Playlists** | Only playlists you marked public |
| **Reposts** | Reposted content, when reposts are enabled |
| **Liked Collections** | Collections you liked |
| **Liked Producers** | Producer profiles you liked |
| **Followers** | People following your user profile |
| **Following** | User accounts you follow |

<Tip>
Use your public profile as a lightweight portfolio of taste. Public playlists, likes, and reposts give other users context about what you are into without duplicating your library pages.
</Tip>

## Linked producer profiles

If you are also a producer:

- Your user profile can link to your producer profile
- Your verified producer badge also appears on the user profile
- Your producer catalog still lives on the producer page, not the user page

<Note>
User profiles and producer profiles are related, but they are edited in different places.
</Note>
