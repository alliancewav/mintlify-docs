# Profile Visibility

BeatPass user profiles are public. This page covers what is visible today and the controls you have.

## What Always Appears

Regardless of what you fill in, visitors can always see:

- Your avatar and username
- A verified badge if your linked producer profile is verified
- Your profile tabs (Liked Beats, Public Playlists, Reposts, Liked Collections, Liked Producers, Followers, Following)

## What the Editor Lets You Save

The profile editor lets you add a first name, last name, city, country, description, and links. However, not all of these fields are visibly displayed on the live user profile page today. The editor saves the data to your account, but visibility on the public page depends on which fields the current profile layout renders.

## What You Can Control

BeatPass does not have a dedicated privacy panel for user profiles. The practical controls available are:

### Leave fields blank

If you do not want certain information stored on your account, leave those editor fields empty.

### Keep playlists private

When a playlist is set to private:

- It stays in your library
- It does not appear on your profile's **Public Playlists** tab

<Tip>
Use private playlists for work-in-progress curation or personal listening. Only make a playlist public when you want it visible on your profile.
</Tip>

## What Visitors Can Do

Visitors to your profile can:

- Follow you
- Send a message when messaging is available for that account state
- Browse your public tabs

<Note>
This page describes controls that exist in the current product. Features like profile-hide toggles or block lists are not available today.
</Note>
