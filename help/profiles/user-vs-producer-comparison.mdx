# User vs Producer Profiles

BeatPass exposes two different public profile types:

- A **user profile** for every account holder
- A **producer profile** for approved producers with an artist page

## Quick Comparison

| Capability | User profile | Producer profile |
| --- | --- | --- |
| **Created automatically with your account** | Yes | No |
| **Used for uploads and catalog management** | No | Yes |
| **Public music catalog** | No | Yes |
| **Can show a verified badge** | Yes, if your linked producer profile is verified | Yes |
| **Editable from your public profile page** | Yes | No |
| **Editable from the producer editor** | No | Yes |
| **Can show gallery images** | No | Yes |
| **Can pin a featured track** | No | Yes |
| **Can show professional credits** | No | Yes |
| **Has follower and messaging actions for visitors** | Yes | Yes |
| **Can link to the other profile type** | Yes, via **Producer Profile** when linked | No |

## User Profile Features

A user profile is your account-facing identity on BeatPass. Every account gets one automatically.

### What visitors can see

- Your avatar and username
- Your first and last name, if you added them
- Your city and country, if you added them
- Your bio and links
- Your public profile tabs

### Public tabs on a user profile

- **Liked Beats**
- **Public Playlists**
- **Reposts**, when reposts are enabled on the platform
- **Liked Collections**
- **Liked Producers**
- **Followers**
- **Following**

### Producer link and verified badge

If your account is linked to a producer profile:

- Your user profile header can show a **Producer Profile** button
- A verified producer badge also appears on your user profile when that producer profile is verified

## Producer Profile Features

A producer profile is BeatPass's public artist page for approved producers.

### What visitors can see

- A cover image and avatar
- Your producer name and verification state
- Your discography and albums
- Gallery images when the gallery feature is enabled
- Professional credits in the discography view
- Followers
- A pinned track when you choose one

### Public tabs on a producer profile

- **Discography**
- **Similar artists**, when BeatPass has enough related data
- **Gallery**
- **Beats**
- **Albums**
- **Followers**
- **Achievements**, for the owner or admins

### Producer-only tools

Approved producers can also:

- Upload tracks and collections
- Edit their producer profile from **More -> Edit**
- Manage credits, images, genres, and biography
- Use analytics, finances, and other producer dashboard tools

## How to Upgrade to a Producer Profile

If you want a producer profile, start in the Producer Program flow. The current request types include:

1. **Become a Producer**
2. **Claim Existing Artist**
3. **Get verified**

<Note>
The exact path depends on whether you are starting fresh, claiming an existing page, or requesting verification for an existing presence.
</Note>

## Which Should I Choose?

**Choose User Profile if you:**
- Primarily listen to and collect beats
- Want to organize beats into playlists
- Don't plan to upload your own music

**Choose Producer Profile if you:**
- Create your own beats
- Want to build an audience for your music
- Plan to license or sell your beats
