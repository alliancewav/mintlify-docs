# User vs Producer Profiles

BeatPass has two public profile types:

- A **user profile** for every account holder
- A **producer profile** for approved producers with an artist page

## Quick Comparison

| Capability | User profile | Producer profile |
| --- | --- | --- |
| **Created automatically** | Yes | No — requires Producer Program approval |
| **Upload and catalog management** | No | Yes |
| **Public music catalog** | No | Yes |
| **Verified badge** | Yes, if linked producer is verified | Yes |
| **Edit from your profile page** | Yes, via **Edit** button | Yes, via **More** → **Edit** |
| **Gallery images** | No | Yes |
| **Pinned featured track** | No | Yes |
| **Professional credits** | No | Yes |
| **Follower and messaging actions** | Yes | Yes |
| **Link to other profile type** | Yes, via **Producer Profile** button | No |

## User Profile

A user profile is your account-facing identity on BeatPass. Every account gets one automatically.

### What visitors see

- Your avatar and username
- A verified badge when your linked producer profile is verified
- Your public profile tabs

<Note>
The profile editor lets you save additional details like name, city, country, description, and links. Not all of these are visibly rendered on the live user profile page today.
</Note>

### Public tabs

- **Liked Beats**
- **Public Playlists**
- **Reposts** (when reposts are enabled)
- **Liked Collections**
- **Liked Producers**
- **Followers**
- **Following**

### Producer link

If your account is linked to a producer profile:

- Your user profile header shows a **Producer Profile** button
- A verified badge also appears when that producer profile is verified

## Producer Profile

A producer profile is BeatPass's public artist page for approved producers.

### What visitors see

- A cover image and avatar
- Producer name and verification status
- Discography and albums
- Gallery images
- Professional credits in Discography
- Followers
- A pinned track when set

### Public tabs

- **Discography**
- **Similar Artists** (when BeatPass has enough related data)
- **Gallery**
- **Beats**
- **Albums**
- **Followers**
- **Achievements** (visible to the owner and admins)

### Producer-only tools

Approved producers can also:

- Upload tracks and collections
- Edit their producer profile from **More** → **Edit**
- Manage credits, images, genres, and biography
- Use analytics, finances, and other Producer Dashboard tools

## How to Get a Producer Profile

Apply through the Producer Program. The available request types are:

1. **Become a Producer** — Start fresh as a new producer
2. **Claim Existing Artist** — Claim an existing artist page
3. **Get Verified** — Request verification for an approved producer profile

## Which Is Right for Me?

**Stay with a user profile if you:**
- Primarily listen to and collect beats
- Want to organize beats into playlists
- Don't plan to upload your own music

**Apply for a producer profile if you:**
- Create your own beats
- Want to build an audience for your music
- Plan to license or sell your beats
