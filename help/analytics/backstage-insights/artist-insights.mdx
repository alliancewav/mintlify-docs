Artist Insights is the broadest Backstage Insights view. It combines activity across one artist profile so you can review catalog performance from a single page.

<Info>
  Direct route pattern: `/backstage/artists/{artistId}/insights`
</Info>

## Header Information

The current artist header can include:

- follower count
- album count
- total plays
- genres

## What Artist Insights Does Not Show

Artist Insights does not currently include the track-only engagement modules such as **Engagement Overview**, **Listen Duration**, or **Completion Breakdown**.
