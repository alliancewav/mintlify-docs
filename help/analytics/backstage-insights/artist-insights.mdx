Artist Insights is the broadest Backstage Insights view. It combines activity across one artist profile so you can review catalog performance from a single page.

## Opening Artist Insights

Open the **Insights** action on any artist profile you manage from Backstage. The page loads with the **This year** date range by default — adjust it with the floating date-range selector if needed.

## Header Information

The current artist header can include:

- Follower count
- Album count
- Total plays
- Genres

<Note>
  The artist insights header does not display a track count. For track-level detail, open the insights page for a specific track or album instead.
</Note>

## What Artist Insights Does Not Show

Artist Insights does not include the track-only engagement modules: **Engagement Overview**, **Listen Duration**, or **Completion Breakdown**. Open a specific [Track Insights](/help/analytics/backstage-insights/track-insights) page for that detail.
