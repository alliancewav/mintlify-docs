Album Insights rolls performance up across the tracks in one album or collection. Use it when you want one page for the release as a whole instead of opening tracks one by one.

## Opening Album Insights

Open the **Insights** action on any album you manage from Backstage. The page loads with the **This year** date range by default — adjust it with the floating date-range selector if needed.

## Header Information

The current album header can include:

- Release date
- Track count
- Total duration
- Genres
- Plays
- Likes

## What Album Insights Does Not Show

- No track-level engagement section (Engagement Overview, Listen Duration, Completion Breakdown).
- Those modules are only available on individual [Track Insights](/help/analytics/backstage-insights/track-insights) pages.
