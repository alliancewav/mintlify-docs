## What BeatPass Currently Exposes

- **Track Insights** for detailed per-track performance.
- **Album Insights** for rolled-up album performance.
- **Artist Insights** for rolled-up artist-level performance.
- A shared date-range control inside the insights layout.

## What To Expect

<AccordionGroup>
  <Accordion title="Default filter">
    Backstage Insights currently opens with a date-range preset that defaults to **This year**.
  </Accordion>
  <Accordion title="Different views show different charts">
    Track insights include extra engagement sections that do not appear on album or artist insight pages.
  </Accordion>
  <Accordion title="Exports">
    The standard producer-facing Backstage Insights UI does not currently show a built-in export button. See [Exporting Analytics Data](/help/analytics/backstage-insights/exporting-data) for the current status.
  </Accordion>
</AccordionGroup>
