The Home page is the landing view for the Producer Dashboard. It is built for orientation: who you are signed in as, what your current artist context looks like, and where to go next.

<Info>
  Open it at <a href="https://open.beatpass.ca/producer-dashboard">https://open.beatpass.ca/producer-dashboard</a>.
</Info>

## What you see first

<CardGroup cols={3}>
  <Card title="Greeting header" icon="hand">
    Time-based greeting, current date, artist or display name, and your current level summary.
  </Card>
  <Card title="Quick Navigation" icon="compass">
    Shortcut cards generated from the current dashboard menu configuration.
  </Card>
  <Card title="Quick Stats" icon="chart-bar">
    Eight at-a-glance cards for catalog size, activity, and profile access.
  </Card>
</CardGroup>

## Greeting header

The top of the page personalizes the dashboard around the signed-in account.

### The header can include

- A greeting such as morning, afternoon, or evening
- The current day and date
- Your linked artist name when available
- Your BeatPass display name as a fallback
- Your current producer level title
- Your current XP total

<Note>
  The level summary comes from the same progress system used on the full [Progress page](/help/producer-dashboard/progress).
</Note>

## Quick Navigation

The shortcut grid is the fastest way to move between dashboard pages.

### How it works

- The cards are built from the same menu configuration used by the dashboard sidebar
- Most cards open other Producer Dashboard pages
- Some cards may point outside the dashboard and open in a new tab
- Card descriptions are more visible on larger screens

### If the grid is empty

BeatPass shows **No menu items configured** instead of fake shortcuts. That means the current site configuration does not have shortcut cards available yet.

## Quick Stats

The Home page uses the **Quick Stats** section to surface the most visible numbers for your current artist context.

| Card | What it tracks |
| --- | --- |
| **Tracks** | Total tracks for the current artist |
| **Total Plays** | Lifetime plays across the current catalog |
| **Albums** | Total album count |
| **Downloads** | Track download count |
| **Followers** | Follower total |
| **Level** | Current level and XP |
| **Growth** | Current growth rate or a calculating/loading state |
| **View Profile** | Shortcut to your public artist page |

<Tip>
  Use the Home page to decide where to investigate next. If a number changes here, open the page that owns that signal instead of treating the Home page like a full report.
</Tip>

## What to expect when data is missing

<AccordionGroup>
  <Accordion title="You are new to the dashboard" icon="sparkles" defaultOpen>
    New artists often see `0` values on several cards until tracks, plays, downloads, or followers begin to accumulate.
  </Accordion>
  <Accordion title="Growth is still calculating" icon="chart-line">
    The **Growth** card can stay neutral while BeatPass finishes calculating the current rate. It does not insert a made-up percentage.
  </Accordion>
  <Accordion title="No linked artist profile yet" icon="user">
    Other dashboard pages will send you to setup or Backstage if the account is missing an artist profile.
  </Accordion>
</AccordionGroup>
