The Progress page is the XP and achievements view of the Producer Dashboard. It shows your level, current XP, remaining XP to the next level, and the achievements you have already unlocked or still have left to earn.

<Info>
  Open it at <a href="https://open.beatpass.ca/producer-dashboard/progress">https://open.beatpass.ca/producer-dashboard/progress</a>.
</Info>

## Level summary

The header is built around your current progress state.

### It includes

- Your current level title
- Your current level number
- Total XP earned so far
- A progress ring toward the next level
- Remaining XP needed to level up

## How XP is currently awarded

The in-app **How XP Works** guide currently lists these base sources:

| Action | Current guide value |
| --- | --- |
| **Upload Tracks** | `+100 XP` per track |
| **Get Plays** | `+1 XP` per play |
| **Complete Metadata** | `+50 XP` per complete track |
| **Collaborate** | `+25 XP` per collaboration |
| **Achievements** | `100-10,000 XP` depending on the achievement |

<Note>
  The guide also explains two important edge cases: collaborative play XP depends on collaborator profile completion, and some special achievements can count toward progress while awarding `0 XP`.
</Note>

## Refresh behavior

The status bar above the XP actions includes:

- The last updated time for the current progress snapshot
- The **How XP Works** button
- A manual **Refresh** button

Use **Refresh** when you want BeatPass to request newer progress data. Do not expect every new event to appear instantly without a refresh.

## Achievements

The lower section tracks both completed and remaining achievements.

### Summary cards

| Card | What it tracks |
| --- | --- |
| **Unlocked** | Achievements already earned |
| **Locked** | Achievements still available |
| **Total XP Earned** | XP received from achievements |

### Achievement list details

Entries can show:

- Achievement name
- Locked or unlocked state
- Rarity styling
- XP reward
- Unlocked time for earned achievements
- Description or requirement text

Locked achievements stay visible so you can see what still counts toward your next milestones.

<Tip>
  Use the [Tracks page](/help/producer-dashboard/tracks) and [Activity page](/help/producer-dashboard/activity) alongside this one. Those pages help you create the uploads, plays, metadata quality, and collaboration signals that turn into XP here.
</Tip>
