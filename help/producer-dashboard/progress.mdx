The **Progress** page is the XP and achievements view of the Producer Dashboard. It shows your level, current XP, remaining XP to the next level, and the achievements you have unlocked or still have left to earn.

<ProducerOnly />

## How to Get Here

Click your **profile avatar** → **Backstage**, then select **Progress** in the sidebar.

## Level Summary

The header is built around your current progress state:

- Your current level title and level number
- Total XP earned so far
- A progress ring showing progress toward the next level
- Remaining XP needed to level up

## How XP Is Awarded

Select the **How XP Works** button in the status bar to open the in-app guide. Current base sources:

| Action | XP |
| --- | --- |
| **Upload Tracks** | +100 XP per track |
| **Get Plays** | +1 XP per play |
| **Complete Metadata** | +50 XP per complete track |
| **Collaborate** | +25 XP per collaboration |
| **Achievements** | 100–10,000 XP depending on the achievement |

<Note>
  Collaborative play XP depends on collaborator profile completion. Some special achievements count toward progress while awarding 0 XP.
</Note>

## Refresh Behaviour

The status bar above the XP actions includes:

- The last updated time for the current progress snapshot
- The **How XP Works** button
- A manual **Refresh** button

Use **Refresh** when you want BeatPass to request newer progress data. New events may not appear instantly without a refresh.

## Achievements

### Summary Cards

| Card | What it tracks |
| --- | --- |
| **Unlocked** | Achievements already earned |
| **Locked** | Achievements still available |
| **Total XP Earned** | XP received from achievements |

### Achievement Details

Each entry can show:

- Achievement name and description
- Locked or unlocked state with rarity styling
- XP reward
- Unlocked time for earned achievements

Locked achievements stay visible so you can see what still counts toward your next milestones.

<Tip>
  Use the [My Beats](/help/producer-dashboard/tracks) and [Activity](/help/producer-dashboard/activity) pages alongside this one. Uploads, plays, metadata quality, and collaboration signals all turn into XP here.
</Tip>
