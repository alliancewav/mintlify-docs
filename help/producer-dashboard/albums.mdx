The Albums page is the album-focused view of the Producer Dashboard. It is meant for overview, filtering, and quick actions across your existing releases.

<Info>
  Open it at <a href="https://open.beatpass.ca/producer-dashboard/albums">https://open.beatpass.ca/producer-dashboard/albums</a>.
</Info>

## Main sections

<CardGroup cols={3}>
  <Card title="Header" icon="compact-disc">
    Current artist context and a compact total-albums summary.
  </Card>
  <Card title="Albums Overview" icon="chart-bar">
    Summary cards for albums, album tracks, plays, and current-month activity.
  </Card>
  <Card title="My Albums table" icon="table">
    Filterable list of album releases with edit and insight shortcuts.
  </Card>

## Albums Overview

| Card | What it shows |
| --- | --- |
| **Total Albums** | Album count plus current-month additions |
| **Album Tracks** | Track totals tied to albums |
| **Total Plays** | Combined plays across the album catalog |
| **This Month** | New album count in the current month |

## My Albums table

The **My Albums** table is the main working area for album management.

### Table columns

| Column | What it shows |
| --- | --- |
| **Album** | Artwork and album link |
| **Artist** | Primary artist |
| **Release date** | Album release date |
| **Track count** | Number of tracks in the album |
| **Plays** | Album play total |
| **Last updated** | Most recent update date |
| **Actions** | Quick links for insights and editing |

### Actions currently available

- **Upload album** opens the Backstage album creation flow
- **Insights** opens the album's Backstage insights page
- **Edit** opens the album editor
- **Delete selected** appears when you select table rows

### Filters currently available

- Artwork
- Plays count
- Created date
- Updated date
- Artist

<Tip>
  Use this page when you need release-level triage. Use the linked insights screen when you need a deeper breakdown for one album.
</Tip>

## When the page is empty

<AccordionGroup>
  <Accordion title="No linked artist profile" icon="user" defaultOpen>
    BeatPass redirects you back to **Backstage** setup before it can load album data.
  </Accordion>
  <Accordion title="No albums created yet" icon="compact-disc">
    The table stays empty until you publish your first album. In that state, the upload action becomes the main next step.
  </Accordion>
</AccordionGroup>
