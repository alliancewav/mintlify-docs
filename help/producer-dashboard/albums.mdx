The **My Collections** page is the album-focused view of the Producer Dashboard. Use it for overview, filtering, and quick actions across your existing releases.

<ProducerOnly />

## How to Get Here

Click your **profile avatar** → **Backstage**, then select **My Collections** in the sidebar.

## Main Sections

<CardGroup cols={3}>
  <Card title="Header" icon="compact-disc">
    Current artist context and album total.
  </Card>
  <Card title="Albums Overview" icon="chart-bar">
    Summary cards for albums, album tracks, plays, and current-month activity.
  </Card>
  <Card title="Albums table" icon="table">
    Filterable list of releases with edit and insight shortcuts.
  </Card>
</CardGroup>

## Albums Overview

| Card | What it shows |
| --- | --- |
| **Total Albums** | Album count plus current-month additions |
| **Album Tracks** | Track totals tied to albums |
| **Total Plays** | Combined plays across the album catalog |
| **This Month** | New albums added in the current month |

## Albums Table

### Columns

| Column | What it shows |
| --- | --- |
| **Album** | Artwork and album link |
| **Artist** | Primary artist |
| **Release date** | Album release date |
| **Track count** | Number of tracks in the album |
| **Plays** | Album play total |
| **Last updated** | Most recent update date |
| **Actions** | Quick links for **Insights** and **Edit** |

### Available Actions

- **Upload album** — opens the album creation flow
- **Insights** — opens the album's backstage insights page
- **Edit** — opens the album editor
- **Delete selected** — appears when you select table rows

### Filters

- Artwork
- Plays count
- Created date
- Updated date
- Artist

<Tip>
  Use this page for release-level triage. Open the linked insights page when you need a deeper breakdown for one album.
</Tip>

## Empty States

<AccordionGroup>
  <Accordion title="No linked artist profile" icon="user" defaultOpen>
    The page shows a not-found state when your account does not have a linked artist profile. Complete your [Producer Program application](/help/producer-program) first.
  </Accordion>
  <Accordion title="No albums created yet" icon="compact-disc">
    The table stays empty until you publish your first album. The **Upload album** action becomes the main next step.
  </Accordion>
</AccordionGroup>
