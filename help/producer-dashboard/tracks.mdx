The Tracks page is your dashboard-level catalog view. It combines summary cards, a searchable **My Tracks** table, growth guidance, and collaboration tools for your current artist.

<Info>
  Open it at <a href="https://open.beatpass.ca/producer-dashboard/tracks">https://open.beatpass.ca/producer-dashboard/tracks</a>.
</Info>

## Header and growth status

The page header summarizes how your catalog is trending.

### Growth labels you may see

- **Exceptional growth!**
- **Strong growth**
- **Growing steadily**
- **Slight decline**
- **Needs attention**

### How the trend is grouped

The growth guide currently explains the broad direction like this:

- Positive growth: above `+5%`
- Stable range: between `-5%` and `+5%`
- Negative growth: below `-5%`

Use the help icon in the header to open the growth guide and read the improvement tips attached to the current state.

## Overview cards

| Card | What it shows |
| --- | --- |
| **Total Tracks** | Catalog size plus recent upload context |
| **Total Plays** | Lifetime plays and recent change |
| **Growth Rate** | Positive, stable, or declining trend |
| **Collaborators** | Unique collaborators and collaborative-track totals |

## My Tracks table

This is the main working area on the page.

### Table columns

| Column | What it shows |
| --- | --- |
| **Track** | Artwork and track link |
| **Artist** | Primary artist |
| **Duration** | Track length |
| **Plays** | Play total |
| **Growth** | Per-track trend indicator |
| **Points** | Contribution points |
| **Last uploaded** | Upload date |
| **Actions** | Quick links for insights and editing |

### Actions currently available

- **Upload new track** opens the Backstage upload flow
- **Insights** opens the track's Backstage insights page
- **Edit** opens the track edit page
- **Delete selected** appears when you select rows

### Filters currently available

- Artwork
- Album
- Plays count
- Date uploaded
- Licensing
- Points

<Note>
  This page helps you manage and review the catalog, but it does not replace the full edit forms in Backstage.
</Note>

## Collaboration Insights

The **Collaboration Insights** section focuses on the people already connected to your catalog.

It can show:

- Total collaborator count
- Collaborative-track count
- A ranked list of top collaborators
- A larger detail modal when more collaboration data is available

## Collaboration Recommendations

The **Collaboration Recommendations** section suggests artists to review next based on the recommendation data BeatPass currently has for you.

### What appears here

- A short explanation panel behind the info button
- Your top genres used as recommendation context
- Suggested artists
- Similarity signals and shared genres
- A **View Profile** action for each suggestion

These are suggestions, not guarantees. If BeatPass does not have enough data yet, the page tells you to build out your catalog instead of inventing recommendations.
