The Producer Dashboard is your central hub for analytics, earnings, gamification, and catalog management on BeatPass. Access it at `/producer-dashboard` after logging in with an artist profile.

<Info>
  **Audience:** Producers with an artist profile (upload permissions required)
</Info>

---

## Dashboard vs Insights

BeatPass provides two complementary analytics systems:

## Accessing the Dashboard

### Requirements

To access the Producer Dashboard, you need:
- A BeatPass account
- An artist profile linked to your account
- Upload permissions enabled

### Navigation

1. Log in to BeatPass
2. Navigate to `/producer-dashboard` or click **Producer Dashboard** in the navigation menu
3. Use the sidebar to access different tools

## Connecting to Backstage Insights

### Producer Tools in Backstage Headers

When viewing any Backstage Insights page, you'll see **Producer Tools** links in the header:

<CardGroup cols={3}>
  <Card title="Track Insights" icon="record-vinyl">
    Links to **Manage Tracks** and **Dashboard**
  </Card>
  <Card title="Album Insights" icon="layer-group">
    Links to **Manage Albums** and **Dashboard**
  </Card>
  <Card title="Artist Insights" icon="user-music">
    Links to **Finances**, **Activity**, **Progress**, and **Dashboard**
  </Card>
</CardGroup>

This integration ensures you can:
- View detailed analytics for a specific item
- Quickly jump to manage your full catalog
- Access financial and progress tools without navigating back
