BeatPass occasionally offers discount codes that reduce the cost of subscriptions or exclusive purchases. This guide explains how to find and apply them.

---

## Where to Find Discount Codes

Discount codes may be shared through:

- **BeatPass emails** — Promotional emails and newsletters
- **Social media** — Official BeatPass accounts on X, Instagram, and Discord
- **Producer promotions** — Some producers share codes with their audience
- **Special events** — Seasonal promotions or platform milestones

<Note>
  Not all plans or purchases are eligible for every discount code. The code will indicate what it applies to when you enter it.
</Note>

---

## Applying a Discount Code

### During Checkout

<Steps>
  <Step title="Open checkout">
    Start the subscription or exclusive purchase checkout where you want to use the code.
  </Step>
  <Step title="Find the code field">
    Look for the field with the placeholder **Enter promo code**.
  </Step>
  <Step title="Enter your code">
    Paste or type the code, then click **Apply**.
  </Step>
  <Step title="Apply and verify">
    BeatPass updates the checkout summary before you pay.
  </Step>
  <Step title="Complete payment">
    Review the discounted total and complete your purchase as normal.
  </Step>
</Steps>

<Note>
  BeatPass automatically converts coupon codes to uppercase during validation.
</Note>

## Types of Discounts

<CardGroup cols={2}>
  <Card title="Percentage Off" icon="percent">
    Reduces the price by a percentage (e.g., 20% off). Applied to the base price before taxes.
  </Card>
  <Card title="Fixed Amount Off" icon="dollar-sign">
    Subtracts a specific dollar amount from the total (e.g., \$5 off).
  </Card>
</CardGroup>

### Duration

Some discount codes apply differently depending on how long they last:

- **One-time** — Discount applies to a single payment only
- **Repeating** — Discount applies for a set number of months
- **Forever** — Discount applies to every future payment on the subscription

The duration is shown after you apply the code during checkout.

---

## One Code At A Time

BeatPass currently allows one coupon per checkout.

If a coupon is already applied, remove it before trying another one.

## Common Questions

<AccordionGroup>
  <Accordion title="My code isn't working">
    - Double-check the code
    - Verify the code hasn't expired
    - Confirm the code applies to the product you're purchasing (subscription vs. exclusive beat)
    - Some codes have a limited number of uses — it may have reached its limit
  </Accordion>
  <Accordion title="Can I use more than one code at a time?">
    Only one discount code can be applied per transaction. If you have multiple codes, choose the one with the best value.
  </Accordion>
  <Accordion title="Can I apply a code after I've already paid?">
    Discount codes must be entered before completing payment. They cannot be applied retroactively. If you just subscribed, contact support — they may be able to help.
  </Accordion>
  <Accordion title="Does the discount apply to renewals?">
    That depends on the code's duration. A one-time code only applies to your first payment. A repeating or forever code will continue to apply as described.
  </Accordion>
  <Accordion title="Will the platform fee still appear?">
    Yes. Checkout still shows the platform fee as a separate part of the total.
  </Accordion>
</AccordionGroup>
