The **Get Verified** option is for producers who already have a working BeatPass producer profile and want the verified badge. Unlike the other producer flows, this is not a normal request form. BeatPass sends you into the Discord verification path.

---

## Who Should Use This

Choose this path if:

- You already have BeatPass producer access
- You want the verified badge on your profile
- You are willing to complete the current Discord verification process

If you do not have producer access yet, use [Become a Producer](/help/producer-program/application-types/become-producer) or [Claim Existing Artist](/help/producer-program/application-types/claim-existing-artist) first.

---

## What Verification Gets You

| Benefit | Description |
|---------|-------------|
| **Verified Badge** | A badge appears on your profile to signal authenticity |
| **Priority Support** | Faster support handling from the BeatPass team |
| **Exclusive Events** | Access to verified-producer opportunities and events |
| **Producer Network** | Additional community and collaboration access |

---

## How Verification Works

Unlike the other producer requests, verification happens through Discord rather than a full form.

<Steps>
  <Step title="Open Get Verified">
    Go to **Backstage** and choose **Get Verified**.
  </Step>
  <Step title="Join Discord">
    Use the BeatPass Discord invite from that page.
  </Step>
  <Step title="Complete the server steps">
    Follow the current Discord verification instructions.
  </Step>
  <Step title="Receive the badge">
    Once approved there, your BeatPass profile gets the verified badge and verified-producer perks.
  </Step>
</Steps>

---

## Why Discord?

BeatPass uses Discord because it supports:

- Direct identity confirmation
- Community onboarding
- Faster back-and-forth with moderators
- Ongoing connection with other producers

---

## Common Questions

<AccordionGroup>
  <Accordion title="Do I need Discord to get verified?">
    Yes. The current BeatPass verification flow is Discord-based.
  </Accordion>

  <Accordion title="How long does verification take?">
    Verification depends on how quickly you complete the Discord steps and how quickly the moderation team can review them.
  </Accordion>

  <Accordion title="What if I am already in the Discord but still not verified on BeatPass?">
    Make sure you completed the required Discord steps. If you did and your BeatPass badge still is not showing, ask a moderator in the server for help.
  </Accordion>

  <Accordion title="Can I get verified without already being a producer?">
    No. Verification assumes you already have an active BeatPass producer profile.
  </Accordion>
</AccordionGroup>
