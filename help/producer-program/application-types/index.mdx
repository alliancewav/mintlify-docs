BeatPass uses three request types for producer access and verification. The right one depends on whether you need a new producer profile, access to an existing one, or only a verified badge.

---

## Which Application Type Is Right for You?

| If you... | Choose... | What happens |
| --- | --- | --- |
| Are new to BeatPass and want to start uploading | **Become a Producer** | A new producer profile is created for you |
| See your music on BeatPass but do not control the profile | **Claim Existing Artist** | Your account requests access to the existing profile |
| Already have a producer profile and want verification | **Get Verified** | You are directed to the Discord verification flow |

---

## Become a Producer

**Best for:** Producers who do not have any profile on BeatPass yet.

This creates a brand-new producer profile linked to your account. You provide your producer name, real name, and a link to your beats for review.

<Card title="Learn More" icon="user-plus" href="/help/producer-program/application-types/become-producer">
  Full details on the Become a Producer request.
</Card>

**What you need:**

- Your producer name
- A public link to your beats
- Your real first and last name

---

## Claim Existing Artist

**Best for:** Producers whose music is already on BeatPass but whose current account does not control that profile yet.

This connects an existing producer profile to your account after BeatPass reviews the claim.

<Card title="Learn More" icon="circle-check" href="/help/producer-program/application-types/claim-existing-artist">
  Full details on the Claim Existing Artist request.
</Card>

**What you need:**

- To search for and select the existing profile
- Proof that you are the rightful owner or representative
- A public link to your beats

<Note>
  You can only claim profiles marked **Available**. If a profile shows **Already Claimed**, contact BeatPass support if you believe it should belong to you.
</Note>

---

## Get Verified

**Best for:** Producers who already have a working producer profile on BeatPass and want the verified badge.

Verification happens through the BeatPass Discord community, not through a traditional request form.

<Card title="Learn More" icon="certificate" href="/help/producer-program/application-types/get-verified">
  Full details on the Discord verification flow.
</Card>

**What you need:**

- An existing BeatPass producer profile
- A Discord account
- Active participation in the Discord community

---

## Comparison

| Feature | Become a Producer | Claim Existing Artist | Get Verified |
| --- | :-: | :-: | :-: |
| Creates new profile | Yes | No | No |
| Links existing profile | No | Yes | No |
| Uses request form | Yes | Yes | No |
| Uses Discord flow | No | No | Yes |
| Discord required | No | No | Yes |
| Existing producer profile required | No | No | Yes |

---

## Not Sure Which to Choose?

Ask yourself:

1. Do you already see your music on BeatPass?
2. Does your account already control that profile?
3. Are you trying to get access, or only add verification?

If you are still unsure, contact BeatPass support before submitting the wrong request.

<Note>
  You cannot submit a new request while another one is **Pending**. If your previous access was **Revoked**, you must contact BeatPass support to submit an appeal for admin review.
</Note>
