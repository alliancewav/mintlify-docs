BeatPass awards XP for several producer actions, and metadata completion is one of them. This page covers the XP tied to metadata quality.

---

## Current XP Values

| Action | Current XP |
|--------|------------|
| Track upload | 100 XP |
| Track play | 1 XP |
| Metadata completion | 50 XP |
| Collaboration | 25 XP |

## What Metadata-Complete Means

BeatPass treats a track as metadata-complete when it has all five of these fields filled in:

- **Description**
- **BPM**
- **Key**
- **Scale**
- **Tags**

When a track meets this check, BeatPass awards 50 XP for metadata completion.

---

## How to Improve Metadata Completion

The easiest ways to hit metadata-complete status are:

- Write a useful description
- Make sure BPM, key, and scale are accurate
- Add relevant tags
- Choose accurate genres to reinforce discoverability

## Why This Matters

Better metadata helps in two ways at once:

1. It makes the beat easier to find and license
2. It qualifies the beat for metadata-complete XP

## Best Practices

<AccordionGroup>
  <Accordion title="Treat metadata as catalog quality, not just XP">
    The biggest benefit of strong metadata is better discovery and clearer buyer understanding.
  </Accordion>

  <Accordion title="Review older tracks">
    Updating older beats is often the fastest way to improve your catalog-wide completeness.
  </Accordion>

  <Accordion title="Focus on accuracy first">
    Wrong BPM, wrong key, or misleading tags are worse than leaving a weak field blank.
  </Accordion>
