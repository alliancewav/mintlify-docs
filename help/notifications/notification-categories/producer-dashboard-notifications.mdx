The **Producer Dashboard** category is available to verified producers.

| Notification | Who can receive it | Trigger |
| --- | --- | --- |
| **New Subscribers** | Verified producers | A new user subscribes to the BeatPass platform |
| **New User Registered** | Verified producers | A new user registers on the platform |
| **Analytics Reports** | Verified producers | A scheduled analytics summary is ready |
| **Payout Updates** | Verified producers | A payout is processed or needs attention |
| **Stripe Connect Reminders** | Verified producers | A reminder about incomplete Stripe Connect setup |

<Note>
  **New Subscribers** is a platform-level subscriber update, not a direct follow or fan-subscription notification for your personal profile.
</Note>
