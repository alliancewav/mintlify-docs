BeatPass browser notifications use browser push support plus a service worker subscription. They are a separate channel from email and in-app notifications.

---

## Requirements

Browser notifications need:

- a supported browser with push notification support
- permission granted for BeatPass notifications
- the **Browser** channel enabled in notification settings

---

## Enabling Browser Notifications

1. Open `/notifications/settings`
2. Turn on **Browser** for one or more notification types
3. Approve the browser permission prompt if it appears
4. Save your preferences

If permission is already denied, BeatPass shows a warning banner in the settings page.

---

## Important Behavior

- Browser notifications are independent from **Email** and **In-App**
- Supported browsers can continue showing browser notifications even when BeatPass is not the active tab
- Exact background delivery behavior still depends on the browser and operating system

---

## If Notifications Are Blocked

If you previously denied permission:

1. Open your browser's site settings for BeatPass
2. Change notifications from **Block** to **Allow**
3. Return to `/notifications/settings`
4. Re-enable the **Browser** toggles you want
