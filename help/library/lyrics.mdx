BeatPass includes a library page for saved lyrics at `/library/lyrics`. It acts like a personal lyrics notebook tied to the tracks you have been working on.

<Info>
  Current page title: **Your lyrics**. The subtitle describes it as **Your personal lyrics notepad for all your beats**.
</Info>

## What You Can Do Here

- Search lyrics by lyric text, track name, or artist name.
- Open a saved lyric and jump back into editing.
- Cue the linked track and open the lyrics panel from the library.
- Delete lyric notes you no longer want to keep.

## What Each Lyric Card Includes

<CardGroup cols={2}>
  <Card title="Track context" icon="music">
    Cover art, track name, artists, and the last updated date.
  </Card>
  <Card title="Preview" icon="comment">
    A short excerpt from the saved lyric text.
  </Card>
  <Card title="Quick actions" icon="gear">
    **Edit Lyrics**, play/cue controls, and delete.
  </Card>
  <Card title="Player handoff" icon="microphone-lines">
    Opening a lyric also opens the BeatPass lyrics editor for the linked track.
  </Card>
</CardGroup>

## How To Start Using It

1. Play a beat.
2. Open the lyrics editor from the player.
3. Write and save lyrics.
4. Return to `/library/lyrics` whenever you want to search or reopen them.

## Empty State Behavior

If you have not saved any lyrics yet, BeatPass prompts you to start from the player and use the microphone icon. On mobile, opening a lyric can also open the full player overlay because the lyrics experience lives there.
