Use this page to compare common release scenarios against the current standard non-exclusive license and the current default exclusive terms shown in BeatPass. If you have already purchased an exclusive license, your own preview and certificate override these summary tables.

<Warning>
  Exclusive rights are configurable per track. Use the **Exclusive License Terms** preview and your generated certificate whenever you need the exact answer for a specific purchase.
</Warning>

## Streaming And Digital Releases

| Use case | Standard non-exclusive | Exclusive purchase |
|----------|------------------------|-------------------|
| Release one finished song on Spotify, Apple Music, YouTube Music, and similar platforms | Yes | Yes |
| Monetize that finished song | Yes | Yes |
| Use the same beat for multiple separate finished songs | No, download it again for each project | Check the exclusive terms |

## Video, Sync, And Promotion

| Use case | Standard non-exclusive | Exclusive purchase |
|----------|------------------------|-------------------|
| Music videos, lyric videos, social clips, and similar promotion for your finished song | Yes | Yes |
| Film, TV, commercials, video games, or other sync-style use | No | Check the exclusive terms |
| Content ID registration | No | Only if the exclusive terms allow it |

## Live Use And Broadcasting

| Use case | Standard non-exclusive | Exclusive purchase |
|----------|------------------------|-------------------|
| Live performance up to **50 attendees** | Yes | Yes |
| Larger live events | No | Check the exclusive terms |
| Broadcast radio or TV use | No | Check the exclusive terms |

## Files, Certificates, And Proof

| Item | Standard non-exclusive | Exclusive purchase |
|------|------------------------|-------------------|
| Audio file delivery | The beat's downloadable source file | The files listed in the exclusive preview |
| Stems or extra formats | Not part of the standard terms | Only if the preview includes them |
| License proof | Certificate stored in **Library** -> **Licenses** | Certificate generated for the purchase |

## Credit, Ownership, And Restrictions

- Standard non-exclusive downloads require producer credit.
- Standard non-exclusive use is tied to **one finished song per download**.
- Both paths prohibit reselling the beat itself as a standalone product.
- Both paths require you to follow the exact certificate or preview attached to your beat.
- Exclusive purchases can still include producer publishing or royalty obligations, so review them before checkout.

## Quick Summary

### Standard Non-Exclusive

- Good for releasing and monetizing one finished song.
- Includes public release and small live use.
- Does not include Content ID, sync use, or large-event rights.

### Exclusive Purchase

- Best when you need the specific expanded rights listed in the track's preview.
- Often includes a broader file package and broader commercial permissions.
- Still requires you to read the actual offer because exclusive terms are not one-size-fits-all.
