This page provides a clear overview of your usage rights when you license beats from BeatPass. Your specific rights depend on whether you have a non-exclusive (subscription) or exclusive license.

## Streaming Platforms

### What You Can Do

| Platform | Non-Exclusive | Exclusive |
|----------|---------------|-----------|
| Spotify | Yes | Yes |
| Apple Music | Yes | Yes |
| YouTube Music | Yes | Yes |
| Amazon Music | Yes | Yes |
| Tidal | Yes | Yes |
| SoundCloud | Yes | Yes |
| Other streaming services | Yes | Yes |

Both license types allow unlimited streams on all major streaming platforms.

### Monetization

- **Non-Exclusive**: You can monetize your releases on all platforms
- **Exclusive**: Full monetization rights plus Content ID registration

## Video Content

### YouTube & Social Media

| Use Case | Non-Exclusive | Exclusive |
|----------|---------------|-----------|
| Music videos | Yes | Yes |
| Lyric videos | Yes | Yes |
| Social media posts | Yes | Yes |
| YouTube monetization | Yes | Yes |
| TikTok/Reels/Shorts | Yes | Yes |

### Professional Video

| Use Case | Non-Exclusive | Exclusive |
|----------|---------------|-----------|
| Television shows | No | Yes |
| Films/Movies | No | Yes |
| Commercials | No | Yes |
| Streaming series (Netflix, etc.) | No | Yes |
| Video games | No | Yes |

<Warning>
  TV, film, commercial, and video game use requires an exclusive license or separate agreement with the producer.
</Warning>

## Live Performance

| Venue Size | Non-Exclusive | Exclusive |
|------------|---------------|-----------|
| Up to 50 attendees | Yes | Yes |
| 51-500 attendees | No | Yes |
| 500+ attendees | No | Yes |
| Festivals | No | Yes |
| Broadcast performances | No | Yes |

<Note>
  The 50-person limit for non-exclusive licenses helps ensure fair use while allowing reasonable performance opportunities.
</Note>

## Physical Distribution

| Format | Non-Exclusive | Exclusive |
|--------|---------------|-----------|
| CDs | Yes | Yes |
| Vinyl records | Yes | Yes |
| USB drives | Yes | Yes |
| Other physical media | Yes | Yes |

Both license types allow physical distribution of your finished song.

## Radio & Broadcasting

| Use Case | Non-Exclusive | Exclusive |
|----------|---------------|-----------|
| Internet radio | Yes | Yes |
| College radio | Yes | Yes |
| Commercial radio | Limited | Yes |
| TV broadcast | No | Yes |

For non-exclusive licenses, radio play is permitted but broader broadcasting rights require an exclusive license.

## Content ID & Copyright Registration

| Action | Non-Exclusive | Exclusive |
|--------|---------------|-----------|
| Register song with PROs | Yes | Yes |
| Content ID (YouTube) | No | Yes |
| Audio fingerprinting | No | Yes |
| Copyright claims against others | No | Yes |

<Warning>
  Non-exclusive licensees cannot register for Content ID because it would block other valid licensees using the same beat.
</Warning>

## Modifications Allowed

Both license types allow you to modify the beat for your project:

| Modification | Allowed |
|--------------|---------|
| Change tempo/BPM | Yes |
| Change key | Yes |
| Rearrange sections | Yes |
| Add or remove elements | Yes |
| Mix with your vocals | Yes |
| EQ and effects | Yes |

<Note>
  With exclusive licenses, you also receive stem files, allowing more detailed modifications.
</Note>

## What's Always Prohibited

Regardless of license type, you cannot:

| Prohibited Action | Reason |
|-------------------|--------|
| Claim you produced the beat | The producer retains credit |
| Sell the beat file to others | Redistribution not allowed |
| Transfer your license | Licenses are personal and non-transferable |
| Use the beat to create sample packs | This would be redistribution |
| Remove producer tags without license | Tags protect the producer |

## Attribution Requirements

### Non-Exclusive License

You **must** credit the producer:

> **"Produced by [Producer Name]"**

Include this in:
- Song metadata
- Streaming platform credits
- Video descriptions

### Exclusive License

Attribution requirements vary by agreement. Check your specific license terms — some exclusive deals may not require attribution.
