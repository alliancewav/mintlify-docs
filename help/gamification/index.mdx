## Quick Reference

### XP Sources

| Action | XP Earned |
|--------|-----------|
| Upload a track | +100 XP |
| Track plays | +1 XP per play |
| Complete metadata | +50 XP per track |
| Collaborate | +25 XP per collaboration |
| Unlock achievements | Variable (150-6,000+ XP) |

### Level Titles

| Level Range | Title |
|-------------|-------|
| 1-2 | Novice Producer |
| 3-4 | Apprentice Producer |
| 5-6 | Intermediate Producer |
| 7-9 | Skilled Producer |
| 10-14 | Experienced Producer |
| 15-19 | Advanced Producer |
| 20-29 | Professional Producer |
| 30-39 | Expert Producer |
| 40-49 | Master Producer |
| 50-100 | Legendary Producer |

### Achievement Rarity

| Rarity | XP Range | Difficulty |
|--------|----------|------------|
| Common | 100-1,000 XP | Entry-level |
| Rare | 1,500-2,000 XP | Consistent effort |
| Epic | 2,500-3,500 XP | Significant accomplishment |
| Legendary | 4,000-6,000 XP | Elite milestone |

### Leaderboard Categories

| Category | Weight in Overall |
|----------|-------------------|
| Points (XP) | 25% |
| Plays | 20% |
| Uploads | 15% |
| Quality | 15% |
| Collaboration | 15% |
| Earnings | 10% |

## How It All Connects

```
Activity → XP → Levels → Achievements → Leaderboard Rankings
   ↑                                            ↓
   └───────────── Recognition & Visibility ─────┘
```

1. **Activities** (uploads, plays, metadata, collaborations) earn XP
2. **XP** accumulates to increase your **Level**
3. Milestones unlock **Achievements** for bonus XP
4. Performance across all metrics determines **Leaderboard** position
5. High rankings increase **Visibility** and discovery

## Getting Started

<Steps>
  <Step title="Check Your Progress">
    Go to `/producer-dashboard/progress` to see your current level, XP, and achievements.
  </Step>
  
  <Step title="Complete Easy Achievements">
    Start with Profile achievements (Photogenic, Oracle, Linked, Model)—they're quick wins for immediate XP.
  </Step>
  
  <Step title="Upload Consistently">
    Each track upload earns +100 XP. Regular uploads also help with upload percentile achievements.
  </Step>
  
  <Step title="Fill Out Metadata">
    Complete all metadata fields on your tracks for +50 XP each and metadata achievements.
  </Step>
  
  <Step title="Check Leaderboards">
    Visit `/producer-dashboard/leaderboards` to see your rank and identify improvement areas.
  </Step>
</Steps>
