## Overview

The Progress Page is your personal producer journey tracker on BeatPass. It displays your XP progress, level status, and all achievements‚Äîboth unlocked and locked‚Äîin one comprehensive view.

<Info>
  **Access:** Navigate to `/producer-dashboard/progress` from your Producer Dashboard.
</Info>

---

## Page Sections

The Progress Page is organized into several key sections:

| Section                | Purpose                                                   |
| ---------------------- | --------------------------------------------------------- |
| **Progress Header**    | Your current level, title, XP, and circular progress ring |
| **Cache Status Bar**   | Data freshness indicator with manual refresh option       |
| **XP Actions Guide**   | Quick ways to earn XP                                     |
| **Achievements Panel** | Stats, progress bar, and full achievement list            |

---

## Progress Header

The hero section displays your producer journey at a glance.

### Elements

| Element            | Description                                                                       |
| ------------------ | --------------------------------------------------------------------------------- |
| **Level Badge**    | Your current level number with "Level X" label                                    |
| **XP Badge**       | Total XP earned displayed in amber pill                                           |
| **Producer Title** | Your rank title (e.g., "Skilled Producer", "Expert Producer")                     |
| **Subtitle**       | "Track your progress, level up, and unlock achievements on your producer journey" |
| **Progress Ring**  | Circular Apple Watch-style progress indicator                                     |
| **Back Link**      | Quick navigation back to Producer Dashboard                                       |

### Progress Ring

The circular progress ring provides an at-a-glance view of your level progress:

- **Outer ring** ‚Äî Background track showing total progress needed
- **Progress arc** ‚Äî Fills based on your percentage to next level
- **Center percentage** ‚Äî Shows exact progress percentage
- **XP to next** ‚Äî Displays remaining XP needed to level up

<Tip>
  The progress ring animates smoothly when your XP changes, giving satisfying visual feedback as you earn XP.
</Tip>

## Cache Status & Controls

A utility bar showing data freshness and providing manual controls.

| Element                    | Function                                             |
| -------------------------- | ---------------------------------------------------- |
| **Updated X seconds ago**  | Shows when progress data was last refreshed          |
| **Auto-refresh countdown** | Shows time until next automatic refresh (within 60s) |
| **How XP Works**           | Opens the XP Guide dialog explaining all XP sources  |
| **Refresh Button**         | Manually refresh your progress data                  |

<Note>
  Progress data is cached for performance. It automatically refreshes every 60 seconds, but you can force a refresh anytime using the refresh button.
</Note>

#### Achievement Item Layout

Each achievement is displayed as a card with:

<Tabs>
  <Tab title="Unlocked Achievements">
    **Visual Styling:**

    - Background color based on rarity (see Rarity Colors below)
    - Colored star icon matching rarity
    - Full opacity and vibrant colors
    - Hover effect: slight brightness decrease

    **Information Displayed:**

    - **Achievement name** ‚Äî Bold title
    - **Rarity label** ‚Äî Uppercase badge (COMMON, RARE, EPIC, LEGENDARY)
    - **Unlock date** ‚Äî "Unlocked X days ago" or "Today"/"Yesterday"
    - **XP earned** ‚Äî Amber pill showing "+X XP" amount
  </Tab>
  <Tab title="Locked Achievements">
    **Visual Styling:**

    - Red-tinted background (`rgba(239, 68, 68, 0.08)`)
    - Red border accent
    - Red lock icon instead of star
    - Muted text colors
    - Hover effect: slightly intensified red

    **Information Displayed:**

    - **Achievement name** ‚Äî Muted color
    - **Rarity label** ‚Äî Muted badge showing original rarity
    - **LOCKED badge** ‚Äî Bright red pill with lock icon
    - **Description** ‚Äî How to unlock the achievement
    - **XP reward** ‚Äî Red-tinted pill showing potential XP
  </Tab>
  <Tab title="Legendary Shimmer">
    **Legendary achievements** (unlocked) feature a special shimmer effect:

    - Subtle golden gradient sweep animation
    - 2.5-second animation cycle
    - Semi-transparent gold highlight
    - Ring glow around the icon (2px amber ring)

    This makes legendary achievements visually distinctive and prestigious.
  </Tab>
</Tabs>

## Achievement Rarity & Colors

Achievements are color-coded by rarity for quick visual identification:

### Rarity Tiers

| Rarity        | XP Range       | Background Color                    | Text Color | Border                 |
| ------------- | -------------- | ----------------------------------- | ---------- | ---------------------- |
| **Common**    | 150-500 XP     | Gray (`rgba(107, 114, 128, 0.15)`)  | Gray       | Gray border            |
| **Rare**      | 800-2,000 XP   | Blue (`rgba(59, 130, 246, 0.15)`)   | Blue       | Blue border            |
| **Epic**      | 2,500-3,500 XP | Purple (`rgba(168, 85, 247, 0.15)`) | Purple     | Purple border          |
| **Legendary** | 4,000-6,000 XP | Amber (`rgba(245, 158, 11, 0.15)`)  | Amber      | Amber border + shimmer |

<Warning>
  **Locked achievements** always display with a red color scheme regardless of their actual rarity. The rarity label still shows the true rarity (e.g., "EPIC") so you know what tier of achievement it is.
</Warning>

## Tips for Maximizing Achievements

<Steps>
  <Step title="Start with Profile Achievements">
    The four profile achievements (Photogenic, Oracle, Linked, Model) are easy to unlock immediately. Complete your profile to earn 1,400 XP right away.
  </Step>
  <Step title="Focus on Metadata Quality">
    When uploading, always fill out all metadata fields. This earns you the +50 XP per track bonus AND progresses your metadata achievements.
  </Step>
  <Step title="Upload Consistently">
    Regular uploads help you climb the upload percentiles. Even 1-2 tracks per week can significantly improve your ranking over time.
  </Step>
  <Step title="Mark Your Calendar for Holidays">
    Set reminders for Halloween (Oct 30 - Nov 1), Christmas (Dec 23-26), New Year (Dec 31 - Jan 2), and Valentine's Day (Feb 13-15) to earn seasonal achievements.
  </Step>
  <Step title="Collaborate Strategically">
    Reach out to other producers for collaborations. Each collaboration counts toward multiple achievements and exposes your music to new audiences.
  </Step>
  <Step title="Check Locked Achievements Regularly">
    Review the locked achievements list to identify which ones you're closest to unlocking, then focus your efforts accordingly.
  </Step>
</Steps>

## Permanent Unlocks

Once you earn an achievement, it's **permanent**:

- Achievements cannot be lost or revoked
- XP from achievements is added permanently to your total
- Even if platform rankings change, earned achievements remain
- Your achievement history shows the date you unlocked each one

---

## Data Refresh

Achievement data is checked and updated at different intervals:

| Check Type                  | Frequency                             |
| --------------------------- | ------------------------------------- |
| **Profile achievements**    | When you update your profile          |
| **Upload achievements**     | When you upload a track               |
| **Play-based achievements** | Periodically during analytics updates |
| **Milestone achievements**  | Daily check based on account age      |
| **Seasonal achievements**   | When uploading during holiday windows |

<Tip>
  If you believe you've met criteria for an achievement but don't see it, try using the **Refresh** button on the Progress page. Achievement checks may take up to 60 seconds to process.
</Tip>
