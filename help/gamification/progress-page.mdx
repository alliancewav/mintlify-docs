The **Progress** page is the main gamification screen in BeatPass. It combines your current level, XP progress, and achievements on one page.

## Getting There

Open your **Producer Dashboard** from the sidebar, then choose **Progress**.

## Page Header

At the top of the page you see:

| Element | Description |
| --- | --- |
| **Level title** | Your current producer level name (e.g. Novice, Apprentice, Rising, and so on) |
| **XP pill** | Your current XP total |
| **Progress ring** | Visual progress toward the next level |
| **XP to next** | How much XP you need before leveling up |

## Page Controls

- **How XP Works** — Opens the XP guide dialog with current XP sources and notes
- **Refresh** — Manually requests fresher progress data

## Page Sections

### Earn XP

A summary of the main actions that award XP:

| Action | XP |
| --- | --- |
| **Upload Tracks** | +100 XP per track |
| **Get Plays** | +1 XP per play |
| **Add Metadata** | +50 XP per track |
| **Collaborate** | +25 XP |

### Achievements

The full achievements section with:

- **Unlocked** and **Locked** achievement counts
- **Total XP Earned** from achievements
- Rarity labels shown on each achievement
- Some achievements can award no XP but still count toward progress

## Empty State

New or inactive producer accounts see an empty state that directs you to upload a track so you can start earning XP.
