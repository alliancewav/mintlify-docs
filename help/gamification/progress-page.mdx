The **Producer Progress** page is the main gamification screen in BeatPass. It combines your current level, XP progress, and achievements on one page.

<Info>
  Current route: `/producer-dashboard/progress`
</Info>

## What You See At The Top

<CardGroup cols={2}>
  <Card title="Level title" icon="shield-check">
    The large header uses your current producer title.
  </Card>
  <Card title="XP pill" icon="bolt">
    Shows your current XP total.
  </Card>
  <Card title="Progress ring" icon="gauge">
    Visual progress toward the next level.
  </Card>
  <Card title="XP to next" icon="chart-line">
    Shows how much XP you still need before leveling up.
  </Card>
</CardGroup>

## Current Page Controls

<AccordionGroup>
  <Accordion title="Updated timer">
    The page shows a recent update timer near the top so you can see how old the current progress snapshot is.
  </Accordion>
  <Accordion title="How XP Works">
    Opens the XP guide dialog with the current XP sources and important notes.
  </Accordion>
  <Accordion title="Refresh">
    Lets you manually request fresher progress data.
  </Accordion>
</AccordionGroup>

## Page Sections

<CardGroup cols={2}>
  <Card title="Earn XP" icon="sparkles">
    A quick summary of the main actions that currently award XP.
  </Card>
  <Card title="Achievements" icon="star">
    The full achievements section with unlocked, locked, and total XP-earned stats.
  </Card>
</CardGroup>

## Achievement Area Behavior

- Unlocked and locked achievements are shown together in one list.
- The section summarizes **Unlocked**, **Locked**, and **Total XP Earned**.
- Rarity labels are shown directly on achievements.
- Some achievements can award no XP but still count toward progress.

## If You See No Producer Data

New or inactive producer accounts can see an empty state that sends you to **Upload Track** so you can start earning XP and populating your progress page.
