The current routed **Producer Dashboard -> Leaderboards** page does **not** expose a timeframe selector.

## What This Means

- You currently switch categories, not date periods.
- You should not expect built-in buttons such as weekly, monthly, or seasonal filters on the routed producer leaderboard page.
- If BeatPass adds timeframe controls to the producer route later, this doc should be updated with the exact labels shown in the UI.
