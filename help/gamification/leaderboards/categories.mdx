The current **Category Filter** on the producer leaderboard page includes these categories.

<CardGroup cols={2}>
  <Card title="Overall" icon="star">
    Your broad ranking across multiple signals. Treat this as the summary ranking, not a single raw metric.
  </Card>
  <Card title="Quality" icon="shield-check">
    Metadata completeness and track-quality preparation.
  </Card>
  <Card title="Uploads" icon="upload">
    Ranking based on how much music you have published.
  </Card>
  <Card title="Plays" icon="music">
    Ranking based on total listening activity.
  </Card>
  <Card title="Points" icon="bolt">
    Ranking based on XP.
  </Card>
  <Card title="Earnings" icon="dollar-sign">
    Ranking based on the earnings or platform-share style data surfaced by the page.
  </Card>
  <Card title="Collab" icon="users">
    Ranking based on collaboration activity.
  </Card>
</CardGroup>

## Important Note About Formulas

Older internal helpers and older docs described exact leaderboard formulas and weights. Those details are not stable enough across the current product to document as fixed public rules. Use the category label itself and the live ranking guide dialog as the best current explanation.
