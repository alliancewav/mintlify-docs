The routed leaderboard experience in BeatPass currently lives at **Producer Dashboard -> Leaderboards** and focuses on category-based producer rankings.

<Info>
  Current route: `/producer-dashboard/leaderboards`
</Info>

## What You See On The Page

<CardGroup cols={2}>
  <Card title="Your Rank card" icon="star">
    Shows your current rank when BeatPass can identify your artist entry in the loaded leaderboard.
  </Card>
  <Card title="How it works" icon="comment">
    Opens the ranking guide dialog from the header.
  </Card>
  <Card title="Category Filter" icon="sliders">
    Switch between the currently available leaderboard categories.
  </Card>
  <Card title="Leaderboard table" icon="table-columns">
    Displays the ranked producers for the selected category.
  </Card>
</CardGroup>

## Current Scope

- The current routed page uses category switching.
- The current routed page does **not** expose a genre selector.
- The current routed page does **not** expose a timeframe selector.

## Current Categories

See [Leaderboard Categories](/help/gamification/leaderboards/categories) for the current category list and plain-language descriptions.
