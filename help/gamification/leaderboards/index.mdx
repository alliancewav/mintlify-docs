The leaderboard page lets you compare your producer ranking against other producers across several categories.

## Getting There

Open your **Producer Dashboard** from the sidebar, then choose **Leaderboards**.

## What You See

| Element | Description |
| --- | --- |
| **Your Rank card** | Shows your current rank in the selected category |
| **How it works** | Opens a guide dialog explaining the current ranking method |
| **Category Filter** | Switch between ranking categories |
| **Leaderboard table** | Displays ranked producers for the selected category |

## Current Scope

- The leaderboard uses category-based switching
- There is no genre selector — rankings are not genre-specific
- There is no timeframe selector — rankings reflect cumulative activity

## Categories

See [Leaderboard Categories](/help/gamification/leaderboards/categories) for the full list and descriptions.
