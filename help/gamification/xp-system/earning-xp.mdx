The current producer experience highlights these XP sources directly in the product.

## Current XP Sources

| Action | Current XP |
|--------|------------|
| Upload Tracks | `+100 XP` per track |
| Get Plays | `+1 XP` per play |
| Complete Metadata | `+50 XP` per track |
| Collaborate | `+25 XP` |
| Achievement unlocks | Varies by achievement |

## What The Product Also Tells You

<AccordionGroup>
  <Accordion title="Achievements can award different XP amounts">
    The XP guide dialog explains that achievement rewards vary and are not all worth the same amount.
  </Accordion>
  <Accordion title="Some achievements can award 0 XP">
    BeatPass can grant special or seasonal achievements that count toward progress without adding XP.
  </Accordion>
  <Accordion title="Collaboration has an extra requirement">
    The current XP guide warns that collaborators must have completed their user profile to receive XP from collaborative track-play activity.
  </Accordion>
</AccordionGroup>

## Best Ways To Grow XP

- Keep uploading tracks consistently.
- Finish your metadata so eligible tracks can earn the metadata bonus.
- Drive real listening activity on your released tracks.
- Work on collaborations when you and your collaborators have complete profiles.
