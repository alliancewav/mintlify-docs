BeatPass awards XP for specific producer actions. Here are the current XP sources shown on the **Progress** page.

## XP Sources

| Action | XP |
| --- | --- |
| **Upload Tracks** | +100 XP per track |
| **Get Plays** | +1 XP per play |
| **Add Metadata** | +50 XP per track |
| **Collaborate** | +25 XP |
| **Achievement unlocks** | Varies by achievement |

## Additional Notes

- **Achievement rewards vary** — Not all achievements are worth the same XP amount
- **Some achievements award 0 XP** — Special or seasonal achievements can count toward progress without adding XP
- **Collaboration XP** — Collaborators must have a completed user profile to receive XP from collaborative track-play activity

## Growing Your XP

- Upload tracks consistently
- Complete your metadata so eligible tracks earn the metadata bonus
- Drive real listening activity on your released tracks
- Collaborate with other producers who have complete profiles
