BeatPass search supports direct result pages plus autocomplete suggestions from the global search field.

## Starting A Search

You can search from the main search bar or, on mobile result pages, from the sticky search field at the top of the page.

## Autocomplete

The current autocomplete opens after you type more than two characters. It can return direct matches for:

- Tracks
- Artists
- Albums
- Playlists
- Profiles
- Tags

Selecting a suggestion takes you straight to that item.

## Search Result Tabs

When you open a full result page, BeatPass builds the tabs dynamically from the result types that actually have matches. The current tabs are:

- **Top results**
- **Tracks**
- **Artists**
- **Albums**
- **Playlists**
- **Profiles**
- **Tags**

<Note>
  There is no dedicated genre results tab in the current search page. Genre browsing happens through genre channels and genre links instead.
</Note>

## What To Expect On Mobile

The search results page keeps a search field pinned near the top on mobile so you can refine the query without leaving the results screen.

## Practical Search Uses

- Search a producer name, then jump into the producer page.
- Search a track title or mood phrase, then narrow with the track tab.
- Search a descriptive phrase and inspect the **Tags** tab if you are looking for a specific vibe label.
