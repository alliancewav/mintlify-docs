Every producer on BeatPass has a public page where you can listen to their beats, explore their albums, and connect with them directly.

## Header Actions

At the top of a producer page you will see:

- **Play** — start playing the producer's catalog.
- **Follow / Following** — tap the heart icon to follow a producer. It changes to a filled heart and reads **Following** once you have followed them. Tap again to unfollow. Followed producers appear in your library under **Your artists**.
- **Message** — open a direct conversation with the producer.
- **More** — additional options such as radio (when available).

### When Is the Message Button Available?

The **Message** button appears on every producer page, but it is not always usable:

- If you are **not signed in**, tapping it will prompt you to sign in first.
- If the producer profile is **unclaimed** (no owner has linked it), messaging is unavailable.
- If you are viewing **your own** producer page, the button does not appear.

## Profile Badges

- **Verified** — the producer has been verified on BeatPass.
- **Unclaimed** — the profile exists but no producer has claimed it yet.

## Tabs

Producer pages can show several tabs. Which ones appear depends on the producer's activity:

| Tab | When It Appears |
| --- | --- |
| **Discography** | Always available — the broadest overview of the producer's releases. |
| **Beats** | Shows all individual tracks. |
| **Albums** | Shows full collections and albums. |
| **Similar artists** | Only when BeatPass has related producers to suggest. |
| **Gallery** | When the producer has uploaded images. |
| **Followers** | Lists the producer's followers. |
| **Achievements** | Visible only to the producer themselves or an admin. |

## Discovering More From a Producer

- Start with **Discography** for a quick overview.
- Use **Beats** to see the full track list.
- Open **Albums** to browse full collections.
- Check **Similar artists** to discover related producers.
