The current BeatPass channel system supports these layouts:

| Layout | Typical use |
| --- | --- |
| **Grid** | Card-style browsing |
| **Track Table** | Track-heavy discovery with more metadata |
| **List** | Compact stacked rows |
| **Carousel** | Horizontal discovery modules |
| **Nested Slider** | A slider presentation inside larger channel structures |
| **Collection Showcase** | Collection-led presentation for album-style content |

## Important Limits

- Not every layout is valid for every content model.
- Nested channels can use a different layout than the parent channel.
- Layout is chosen by channel configuration; users should not expect a universal layout switcher on every channel page.
