BeatPass browse pages can use different visual layouts depending on what they display. You may see any of the following:

| Layout | What it looks like |
| --- | --- |
| **Grid** | Card-style browsing with cover art and titles |
| **Track table** | A detailed table with metadata columns — common for beat-heavy pages |
| **List** | Compact stacked rows |
| **Carousel** | A horizontal row you can scroll through |
| **Slider** | A sliding presentation inside larger discovery pages |
| **Collection showcase** | An album or collection-focused layout with prominent cover art |

## Good to Know

- Not every layout appears on every type of page. The layout matches the content being shown.
- Pages that contain sub-sections can use a different layout for each section.
- You will not find a layout switcher — the layout is chosen by the page design, not by the user.
