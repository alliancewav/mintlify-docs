BeatPass collection pages are the album-style pages for grouped releases.

## What A Collection Page Can Show

The current collection page header supports:

- Cover image
- Collection title
- Artist avatars and links
- Track count
- Total duration
- Release date

Below the header, a collection page can also include:

- The track table
- An **About this collection** description block
- Tag chips
- A comments section when comments are enabled
- Recommendation modules

## Using A Collection Page

- Use the main play control to queue the collection.
- Start from a specific track row if you do not want to begin at the top.
- Open tags or artist links to continue discovering adjacent content.

## Important Notes

- Some collections may be empty and show the album empty state until tracks are available.
- Comment availability depends on the current BeatPass comment settings and your account permissions.
