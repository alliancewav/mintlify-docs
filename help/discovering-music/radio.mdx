Radio is the continuous-play discovery view built from a single seed.

## Supported Seed Types

The current radio page supports these seeds:

- **Artist**
- **Track**
- **Genre**

<Warning>
  Playlist radio is not part of the current radio route set.
</Warning>

## What The Radio Page Shows

When BeatPass builds a radio page, it currently includes:

- A title in the form `Seed Name Radio`
- A play action for the recommendation set
- Beat count and total duration stats
- A **Recommended Beats** section with the generated track table

## Where Radio Starts

Depending on the page and the current instance settings, you may reach radio from:

- Producer page actions
- Track context or track-page actions
- Genre channel headers

## When To Use Radio

- Start from a **track** when one beat has the exact vibe you want more of.
- Start from an **artist** when you want sounds around a producer's catalog.
- Start from a **genre** when you want a broader style-based session.
