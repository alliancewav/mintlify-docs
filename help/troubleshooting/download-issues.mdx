If downloads are not working, start by checking your account state and the track's licensing state.

## I Cannot See the Download Button

<AccordionGroup>
  <Accordion title="You need a paid plan" icon="credit-card" defaultOpen>
    Downloads require a paid subscription. Free accounts can stream and save music but cannot download. If you click **Download** without the right permission, BeatPass opens an upgrade prompt.
  </Accordion>
  <Accordion title="You are not signed in" icon="right-to-bracket">
    BeatPass requires an authenticated account for download actions. Sign in first.
  </Accordion>
  <Accordion title="The track is not downloadable" icon="ban">
    The **Download** button is hidden when a track is exclusive-only, sold exclusively, or otherwise not available through the standard download flow.
  </Accordion>
</AccordionGroup>

## The Download Fails

<AccordionGroup>
  <Accordion title="Your subscription payment is overdue" icon="triangle-exclamation" defaultOpen>
    BeatPass blocks downloads when your subscription is overdue. You see the message: **"Your subscription payment is overdue. Please resolve your payment to download tracks."** Resolve the billing issue in **Billing** from the avatar menu, then retry.
  </Accordion>
  <Accordion title="The track was sold exclusively" icon="lock">
    If a track has been sold exclusively, the download is blocked with: **"This track has been sold exclusively and is no longer available for download."** Only the owner and exclusive buyer can still download it.
  </Accordion>
  <Accordion title="The browser download never starts" icon="globe">
    Check popup or download blocking in your browser settings, clear browser cache, and try another modern browser.
  </Accordion>
</AccordionGroup>

## Exclusive Tracks Will Not Download

<AccordionGroup>
  <Accordion title="The track is exclusive-only" icon="crown" defaultOpen>
    Exclusive-only tracks are not part of the standard subscription download flow. An info modal appears explaining that the track requires an exclusive purchase.
  </Accordion>
  <Accordion title="The track was already sold exclusively" icon="cart-shopping">
    Once sold, the track is blocked for all standard downloads. Only the owner and the exclusive buyer can still access it.
  </Accordion>
</AccordionGroup>

## I Downloaded a Track but Cannot Find My License

Licenses are auto-generated when you download a track as a non-owner. Open **Your library** → **Licenses** to find the certificate.

If you do not see the entry:

- Refresh the page
- Confirm the download actually completed
- If you are a track owner, no license is generated for self-downloads

## File Format

BeatPass serves the original uploaded audio file. Downloads are not a format bundle — you receive the single file the producer uploaded (typically WAV). There is no format picker during the standard download flow.

## I Downloaded the Wrong Track

Download the correct track instead. If you believe a platform error caused the issue, contact **contact@beatpass.ca** with the track URL and account email.
