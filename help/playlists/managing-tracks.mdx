Track management happens in two places: while you browse the catalog and on the playlist page itself.

## Adding Tracks

The normal add flow is:

1. Open a track context menu.
2. Choose **Add to playlist**.
3. Pick the target playlist, or create a new one.

If you create a new playlist from that flow, the track you started from becomes the first item in the playlist.

## What The Playlist Page Shows

A playlist page currently includes:

- Header details such as title, description, editor avatars, track count, and total duration
- The playlist track table
- A search field labeled **Search within playlist**
- The main play button and follow/unfollow state

If the playlist has no tracks yet, BeatPass shows the empty-state message: `This playlist does not have any tracks yet`.

## Removing Tracks

Track removal is permission-based:

| Role | Can remove tracks |
| --- | --- |
| Owner | Yes |
| Accepted editor on a collaborative playlist | Yes |
| Follower or ordinary viewer | No |

## Searching Inside A Playlist

Use **Search within playlist** on the playlist page when you want to narrow a long track list without leaving the playlist.

## Playback From A Playlist

- Use the main play control to queue the playlist.
- Start from a specific row if you want playback to continue from that point.
- Use the player queue if you want temporary playback changes without editing the playlist itself.

<Tip>
  If a playlist is part of a real project, keep the saved order clean and use the queue for one-off listening changes.
</Tip>
