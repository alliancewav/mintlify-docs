BeatPass producer payouts use **Stripe Express**, not the regular Stripe dashboard many businesses use for direct payments.

## Stripe Express vs. Regular Stripe

| | **Stripe Express** | **Regular Stripe** |
|---|---|---|
| **Used for BeatPass payouts** | Yes | No |
| **Login URL** | [connect.stripe.com/express_login](https://connect.stripe.com/express_login) | [dashboard.stripe.com](https://dashboard.stripe.com) |
| **Typical sign-in method** | Verification code | Email and password |
| **Created through BeatPass onboarding** | Yes | No |

<Warning>
  If you are trying to see BeatPass earnings, do not use `dashboard.stripe.com` unless Stripe specifically directs you there for another account. BeatPass payout access is handled through Stripe Express.
</Warning>

## How to Sign In

<Steps>
  <Step title="Open Stripe Express login">
    Go to [connect.stripe.com/express_login](https://connect.stripe.com/express_login).
  </Step>
  <Step title="Use the same email">
    Enter the email address tied to your BeatPass Stripe Connect setup.
  </Step>
  <Step title="Complete verification">
    Stripe sends a code so you can sign in without a traditional dashboard password.
  </Step>
</Steps>

## What BeatPass Handles vs. What Stripe Handles

### Inside BeatPass

Use **Backstage** → **Finances** for:

- Setup prompts
- Payout readiness signals
- Account overview
- Transaction filters and statuses

### Inside Stripe Express

Use Stripe Express for things like:

- Balance and payout views managed by Stripe
- Payout destination details Stripe allows you to manage
- Tax documents if Stripe generates them for your account

<Note>
  BeatPass currently does not expose a dedicated **View Stripe Dashboard** button in the producer finances UI. Use the direct Stripe Express login page when you need the Stripe-side dashboard.
</Note>

## Common Questions

<AccordionGroup>
  <Accordion title="I already have a regular Stripe account. Is that the same one?">
    No. BeatPass payouts are tied to the Stripe Express account created through BeatPass onboarding.
  </Accordion>

  <Accordion title="Why do I not see my BeatPass earnings at dashboard.stripe.com?">
    That is the wrong dashboard for BeatPass payouts. Use Stripe Express instead.
  </Accordion>

  <Accordion title="What if Stripe says no account was found?">
    Double-check the email address you used during BeatPass onboarding. If setup was never completed, your Stripe Express account may not be ready yet.
  </Accordion>

  <Accordion title="Where do I finish incomplete setup?">
    Go back to **Finances** inside BeatPass. That is where BeatPass shows remaining setup prompts and payout-readiness warnings.
  </Accordion>
</AccordionGroup>
