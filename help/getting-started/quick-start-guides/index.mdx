## Need Help Choosing?

- Start with **Listener Quick Start** if you are browsing and saving music
- Start with **Subscriber Quick Start** if you already have, or want, a paid plan
- Start with **Producer Quick Start** if you are an approved producer or preparing for producer workflows

<Info>
  One account can span multiple roles. For example, you can be both a paid subscriber and an approved producer.
</Info>
