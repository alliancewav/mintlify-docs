BeatPass supports three main account states in practice:

- A free account
- A paid subscriber account
- An approved producer account

You can also be more than one of these at the same time, such as a producer who also subscribes.

## Free Accounts

Every new account starts here.

With a free account, you can:

- Stream the catalog
- Like tracks and collections
- Create playlists
- Follow producers and other users
- Comment where commenting is available
- Browse listening history

<Info>
Downloads and license certificates are paid features.
</Info>

## Paid Subscribers

Paid plans add licensed download access on top of the free-account features.

Subscribers can:

- Download eligible beats under BeatPass's standard subscription licensing
- Access license certificates in **Library -> Licenses**
- Keep the licenses they obtained while subscribed
- Use plan-based premium features such as beat request tokens

### Current Paid Plan Family

BeatPass currently offers **Classic**, **Plus**, and **Pro** paid plans.

<Note>
Use [Plans & Pricing](/help/plans-and-pricing/index) for the current feature matrix, token counts, and plan positioning.
</Note>

## Approved Producers

Approved producers get a public producer profile plus producer-only tools.

Producer access includes:

- Uploading tracks and collections
- Editing the producer profile
- Producer Dashboard access
- Analytics and progress tracking
- Finances and Stripe Connect onboarding

### Producer Dashboard

The current producer dashboard routes include:

| Area | Purpose |
| --- | --- |
| **Home** | Producer overview |
| **Progress** | XP, levels, and achievement progress |
| **Tracks** | Track management |
| **Activity** | Producer activity analytics |
| **Albums** | Collection management |
| **Leaderboards** | Producer rankings |
| **Finances** | Payout setup and financial activity |

BeatPass may expose this through a menu entry labeled **Producer Dashboard** or another custom label, but the dashboard itself lives at `/producer-dashboard`.

## Comparison

| Capability | Free account | Paid subscriber | Approved producer |
| --- | --- | --- | --- |
| Stream the catalog | Yes | Yes | Yes |
| Like tracks and collections | Yes | Yes | Yes |
| Create playlists | Yes | Yes | Yes |
| Download licensed beats | No | Yes | Yes, if subscribed or otherwise eligible |
| Access license certificates | No | Yes | Yes, when downloads are eligible |
| Upload tracks | No | No | Yes |
| Producer profile | No | No | Yes |
| Producer finances and analytics | No | No | Yes |

## Upgrading

### From Free Account to Paid Subscriber

Open the [pricing page](https://open.beatpass.ca/pricing) or use **Billing** from the avatar menu.

### From User to Producer

Use the [Producer Program](/help/producer-program/index) request flow to become a producer, claim an existing artist page, or request verification.
