BeatPass navigation can vary a little by account type, device size, and custom menus. This guide focuses on the stable areas that matter in the current app.

## Core Areas

### Discovery

Use discovery surfaces to move through:

- The homepage and channel-driven recommendations
- Search
- Producer pages
- Track pages
- Collections and playlists

### Library

Your library is where saved and account-linked content lives. Depending on your account state and feature access, common sections include:

- **Your tracks**
- **Your collections**
- **Your producers**
- **Listening history**
- **Licenses**
- **Beat requests**
- **Lyrics**

### Playlists

Playlists are available from the library and playlist surfaces. Public playlists can also appear on your user profile.

### Producer Tools

If you are an approved producer, you also use:

- **Producer Dashboard** at `/producer-dashboard`
- **Upload flow** at `/backstage/upload`

## Top Bar

The top navigation area gives you access to:

- **Search**
- **Notifications**
- **Your avatar menu**

## Avatar Menu

The stable items in the avatar menu are:

| Menu item | What it does |
| --- | --- |
| **Profile page** | Opens your public user profile |
| **Billing** | Opens your subscription and payment settings |
| **Dark mode / Light mode** | Changes theme when theme switching is enabled |
| **Log out** | Signs you out |

Additional items can appear if BeatPass has custom menu entries or producer-specific shortcuts configured.

## Player

Playback controls stay attached to the interface while you browse:

- Use the player bar for play, pause, next, previous, queue access, and volume
- Open lyrics when available on the current track
- Open the queue to see what is playing next

## Keyboard Shortcuts

The currently verified shortcuts are:

| Shortcut | Action |
| --- | --- |
| `Space` | Play or pause |
| `K` | Play or pause |
| `Ctrl` + `ArrowRight` | Next track |
| `Shift` + `ArrowRight` | Next track |
| `Ctrl` + `ArrowLeft` | Previous track |
| `Shift` + `ArrowLeft` | Previous track |

## Mobile Navigation

On mobile, BeatPass compresses navigation into a smaller layout:

- Discovery and search stay accessible from the mobile header
- The player expands into a larger mobile playback view
- Menus and navigation surfaces collapse to fit the screen

## Theme

If theme switching is enabled for your account, use the avatar menu to switch between dark and light modes.
