An exclusive BeatPass license is a one-time purchase for a track that is being offered for exclusive sale. After the purchase completes, no new standard non-exclusive licenses are issued for that beat.

<Info>
  Exclusive purchases are separate from subscription download access and currently use Stripe card checkout.
</Info>

## What exclusive changes

<CardGroup cols={2}>
  <Card title="Before purchase" icon="music">
    The track can still be listed as available for exclusive sale, standard download, or both depending on the producer's setup.
  </Card>
  <Card title="After purchase" icon="crown">
    BeatPass marks the track as sold, stores the exclusive certificate, and blocks new non-exclusive licenses for other users.
  </Card>
</CardGroup>

## Non-exclusive vs exclusive

| Topic | Non-exclusive | Exclusive |
| --- | --- | --- |
| Other artists can still license it | Yes | No new licenses after purchase |
| Content ID registration | No | Allowed under current default terms |
| Film/TV/broadcast/commercial/video game use | No | Allowed under current default terms |
| Live performance cap | 50 attendees | No standard cap in current default terms |
| Transferable | No | Not by default |
| Final terms source | Standard certificate | Purchase preview and generated certificate |

## What to review before paying

The most important step is reading the exclusive terms preview for that track. That preview is where BeatPass shows the current terms that will be used for the purchase.

### Under the current default model, the terms can include

- Worldwide territory
- Perpetual duration
- Producer publishing share `20%`
- Licensee publishing share `80%`
- Producer revenue royalty `3%`
- Content ID registration allowed
- Transfer disabled by default

<Warning>
  Treat the preview and final certificate as the source of truth. Older docs that promised the same rights or deliverables for every exclusive sale were too broad.
</Warning>

## What happens after purchase

<AccordionGroup>
  <Accordion title="The track becomes sold" icon="tag" defaultOpen>
    BeatPass marks the track as sold so other users cannot start new standard downloads for it.
  </Accordion>
  <Accordion title="Your certificate is generated" icon="file-certificate">
    The exclusive license is added to **Library** -> **Licenses**.
  </Accordion>
  <Accordion title="An immediate track download is attempted" icon="download">
    The success flow attempts to download the track file right away.
  </Accordion>
  <Accordion title="Older licenses stay valid" icon="history">
    If other users licensed the beat before the exclusive sale, those earlier licenses remain valid.
  </Accordion>
</AccordionGroup>
