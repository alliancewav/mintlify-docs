BeatPassID is a background audio-analysis process that runs when a track is uploaded to BeatPass. It generates a unique fingerprint for each track and attaches that data to license certificates and verification tools.

<Info>
  BeatPassID runs automatically in the background. You do not need to wait for it or check its status. Your license certificate is generated the moment you download a track.
</Info>

## What BeatPassID does

When a producer uploads a track, BeatPass analyzes the audio and creates a fingerprint. That fingerprint data is then included on license certificates and public verification pages to help confirm which audio file a license covers.

## Where fingerprint data appears

- License certificates
- The license certificate modal
- Public verification pages
- Producer upload and track-management tools

## Fingerprint statuses

| Status | What it means |
| --- | --- |
| **Pending** | BeatPass is still analyzing the track audio |
| **Completed** | Fingerprint data has been added to certificate tools |
| **Failed** | BeatPass could not complete the analysis for this track |

## What fingerprint details can include

When fingerprint analysis is complete, certificates and verification pages can display:

- Fingerprint status
- Fingerprinted date
- Analyzed duration
- Fingerprint hash

## Good to know

<AccordionGroup>
  <Accordion title="Certificates are created on download" icon="file-lines" defaultOpen>
    Your license certificate is generated automatically when you download a track. BeatPassID enriches the certificate with fingerprint metadata, but it does not block certificate creation.
  </Accordion>
  <Accordion title="Verification details" icon="shield-halved">
    Completed fingerprint data lets BeatPass display fingerprint information on the certificate and public verification pages.
  </Accordion>
  <Accordion title="Producer workflows" icon="music">
    Producers see fingerprint status during upload and catalog management as part of track-processing workflows.
  </Accordion>
</AccordionGroup>

## Common questions

<AccordionGroup>
  <Accordion title="My certificate seems to be missing — what should I do?" icon="download" defaultOpen>
    Certificates are generated when you download a track. If you cannot find yours, go to **Your library** → **Licenses** and search for the track. If it still does not appear, contact support at contact@beatpass.ca.
  </Accordion>
  <Accordion title="What if fingerprint status shows failed?" icon="triangle-exclamation">
    A failed fingerprint does not prevent you from accessing your license certificate. It means the enrichment data could not be added. The track may be reprocessed by BeatPass automatically.
  </Accordion>
  <Accordion title="Does BeatPassID change the audio file I download?" icon="wave-square">
    No. BeatPassID is metadata about the track analysis, not a modification to the downloaded audio file.
  </Accordion>
</AccordionGroup>
