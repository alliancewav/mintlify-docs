## April 2020 - Version 2.4.5

### Summary

- Large feature release adding GDPR support, advanced uploads, and new channel tools.

### New Features

- New channel auto update methods for popular artists and most recently added tracks.
- Separate pages for editing a user's own tracks and albums outside the admin area.
- Tag based listing pages for tracks and albums.
- Configurable GDPR cookie notice and confirmation policies on registration.
- Chunked and resumable uploads for large files, with support for XSendFile and XAcceleratedResponse to reduce server load.

### Enhancements

- Double click video to toggle fullscreen in the mini player.
- Music, video, avatars, and other media can be stored on cloud providers such as S3, DigitalOcean, and Backblaze.
- Automatic channel title and SEO placeholder updates when site name changes.
- Better search results when editing channels and manual channel content update options.
- Improved tag and genre suggestion behavior in edit forms.

### Fixes

- Numerous UI, caching, SEO, and playlist handling improvements across BeatPass 2.4.x, including better validation error visibility, scrolling behavior, sitemap generation, permission handling, and history or chart display reliability.
