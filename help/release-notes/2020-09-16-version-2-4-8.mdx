## September 2020 - Version 2.4.8

### Summary

- Permissions, lyrics extraction, and UX improvements for mobile and embeds.

### Enhancements

- Automatic lyrics extraction from uploaded metadata.
- New permissions for embedding tracks and playing audio.
- Mobile account item appears by default in controls.
- Track lists queue entire lists in the player, similar to track tables.
- Better edit layouts on mobile and playlist ordering behavior.
- Improved fullscreen local video handling, dark mode scrollbars, and updated data tables.

### Fixes

- Queue updates correctly in fullscreen.
- Track list sorting, liked track filters, and media file deletion work as expected.
- Homepage channels, uploads, scrolling, and notification URLs are more reliable.
- Many smaller bug fixes across navigation and layout.
