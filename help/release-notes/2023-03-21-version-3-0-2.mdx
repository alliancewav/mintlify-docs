## March 2023 - Version 3.0.2

### Summary

- UI polish and a set of fixes for search, tables, and embeds.

### Changes

- Centered play button on artist grid items.
- Like and repost buttons are disabled instead of removed when viewing your own tracks.

### Fixes

- Mobile song search reliability improved.
- Pricing tables show correctly on landing pages.
- Long track names no longer break player layouts.
- Overlays close correctly on navigation.
- Track list and channel grid playback issues resolved.
- Tag manager and indexing issues fixed for analytics and search.
- Better behavior for selection and external links in menus and tables.
