## May 2023 - Version 3.0.6

### Summary

- Security-focused release with two factor authentication, account controls, and UX improvements.

### What's New

- Time based one time password two factor authentication for BeatPass accounts.
- Admins can suspend or ban users from the admin area.
- Option to show full size social login buttons on authentication pages.
- Ability to block registrations from specific email domains.
- Active sessions panel in account settings.

### Fixes and Improvements

- More responsive tables across the site.
- Better handling of uploads and validation when creating albums with multiple tracks.
- Fixes for playlist loading on mobile and library menus on tablets.
- Visual and UX fixes on checkout and theme handling, including dark mode defaults.
- Compatibility updates for newer search methods and safer admin user handling.
