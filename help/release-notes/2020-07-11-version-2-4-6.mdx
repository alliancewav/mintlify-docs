## July 2020 - Version 2.4.6

### Summary

- Single device login controls and a framework update for BeatPass.

### New Features

- Option to allow only one logged in device per user account at a time.

### Enhancements

- Updated Laravel requirement so BeatPass now needs at least PHP 7.2.5.
- Modernized auth page designs and better admin debugging for server errors.
- Improved uploads, cloud storage support, unified email templates, and better general settings validation.
- Landing page toggle, Google Analytics JSON credentials, embed link handling, and mobile search UX improvements.

### Fixes

- Local and YouTube search, genre name handling, playlist ownership display, track SEO fallbacks, cache method validation, cache clearing, popularity sorting, text logo overflow, user background removal, custom CSS or JS storage, and SEO tag placeholder handling.
