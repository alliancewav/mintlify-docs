## March 2023 - Version 3.0.3

### Summary

- Bugfix release focusing on comments, search, and library behavior.

### Fixes

- Added missing translations to localization tools.
- Comments now respect settings and visibility flags.
- Search section links and artist album editing reliability improved.
- User profile dialogs render correctly on mobile.
- Library add or remove actions update without requiring a page refresh.
- Share playlist, channel title translations, and several smaller issues fixed.
