## April 2023 - Version 3.0.5

### Summary

- Introduces HLS and DASH streaming support and a round of stability fixes.

### What's New

- HLS and DASH streams can now be used as track sources.
- Uploaded tracks can inherit genres, tags, and artists from their album.
- Table responsiveness improvements across BeatPass.

### Fixes

- Added missing permissions and fixed role name translations.
- Automation disable behavior is now more reliable.
- Menus now appear correctly on landing pages and in artist discography pagination.
- Pricing savings calculation and several smaller layout issues are resolved.
