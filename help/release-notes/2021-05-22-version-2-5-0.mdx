## May 2021 - Version 2.5.0

### Summary

- Backend tools for importing content and managing channels more reliably.

### Enhancements

- Artists and albums can be imported by Spotify ID even when providers are set to local.

### Fixes

- Manual content channels now work correctly when creating new channels.
- Manual reordering of channel contents is more stable.
- Artist and track names show correctly in browser tab titles.
- SEO settings in the appearance editor save reliably.
- Profile editing and search record importing in settings are more robust.
