## May 2021 - Version 2.5.1

### Summary

- Stability and compatibility update, especially for PHP 8 and search providers.

### Enhancements

- More descriptive error messaging when PHP version is incompatible.
- Artist verified status can be toggled from the admin area.
- Tracks always fall back to a simple seekbar when waveform is unavailable.

### Fixes

- Channel search works reliably when content type is set to multiple types.
- Redirects after creating artists behave correctly.
- Social sharing images for tracks and albums now show consistently.
- Similar artist links, become artist menus, and backstage artist search are fixed.
- Mobile library pages, sitemap generation, Spotify import behavior, PHP 8 radio behavior, and Algolia integration are more reliable.
- Public images stored on S3 are no longer accidentally uploaded as private.
