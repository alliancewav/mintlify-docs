## April 2023 - Version 3.0.4

### Summary

- Quality of life improvements for search and track context actions.

### What's New

- Track, album, artist, and playlist items in search dropdown now have context menus.

### Fixes

- Release date extraction from uploaded files is more accurate.
- Become author menu items now appear on mobile.
- History page no longer breaks when ad blockers are present.
- Better slug generation for artist names and stable search profiles tab behavior.
- Email verification, lyrics buttons, and channel cache invalidation issues resolved.
