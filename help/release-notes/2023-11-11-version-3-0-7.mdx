## November 2023 - Version 3.0.7

### Summary

- Performance, SEO, and admin UX improvements across BeatPass plus a large batch of bug fixes.

### What's New

- Faster first page load for listeners.
- Improved SEO tags editor in the admin area.
- Tags are now searchable for artists and albums.
- New reported filter for comments management.
- Better visibility of user suspension reasons and email verification flows.

### Fixes

- Reliability fixes for HLS playback and iOS video behavior.
- Corrected album and artist links and improved edit flows after claiming artists.
- Improved datepicker behavior and keyboard input.
- Appearance editor persistence and logo handling fixes.
- Fixed issues with subscription plans, PayPal subscriptions, and API access permissions.
- Mobile search and overlay behavior improvements.
- Many smaller fixes for layout, dialogs, and device-specific quirks.
