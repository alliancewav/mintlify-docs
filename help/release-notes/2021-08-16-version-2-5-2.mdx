## August 2021 - Version 2.5.2

### Summary

- Admin quality of life improvements and reliability fixes around comments, channels, and permissions.

### Enhancements

- Comments table in admin now shows the album or track each comment belongs to.
- Channel cache is cleared automatically when a channel is edited or auto updated.
- Spotify IDs can be entered manually for artists, albums, and tracks.
- Updated admin datatable design with additional filters.

### Fixes

- Artist table can be sorted by album count.
- Users can reliably delete their own comments and manage album reposts.
- Queue behavior on radio pages and album to queue behavior are corrected.
- Backstage artist link removal and social login panel visibility now respect permissions.
- SEO tags for albums, interface settings tabs, embed permissions, try pro visibility, and playlist collaboration rules now behave correctly.
