## March 2023 - Version 3.0.1

### Summary

- Targeted bugfix release for search, uploads, channels, and menus.

### Fixes

- Space character works correctly in mobile search.
- Correct track is played when it appears both individually and as part of an album.
- Upload progress shows when uploading multiple tracks during album creation.
- Default genre channels are created correctly during installation.
- Local plus Spotify combined search now functions as expected.
- Channel deletion and channel URLs in the menu manager are fixed.
- Long track names no longer push options buttons off screen.
