## May 2021 - Version 2.4.9

### Summary

- Collaboration, backstage, and API era for BeatPass 2.x.

### New Features

- Collaborative playlists.
- Backstage section for claiming, verifying, or becoming an artist, plus admin tools to manage requests.
- Multiple search integrations including Meilisearch and Elasticsearch for faster, more accurate search.
- Combined Spotify plus local search provider.
- New channel auto update method for Spotify playlists.
- Comments management page in the admin area.
- Import artists, albums, tracks, and playlists from Spotify by ID.
- Embeddable playlists and Spotify genre radio.
- Full token based API for integrating BeatPass with other apps, plus API access token management and live API docs.

### Enhancements and Fixes

- Verified artist badges and richer track tables with artwork.
- Multi artist albums, comment deletion from track pages, and updated Laravel and Angular versions.
- Higher comment length limits and simplified music permissions.
- Profile page customization, history behavior, lyrics provider additions, country and city metadata, and better embeds.
- Numerous fixes for Spotify channels, forms, waveform behavior, embeds, installation, playlist context menus, GDPR embeds, upload behavior, artwork display, navigation warnings, waveform seeking, and more.
