## Overview

Your account settings control how you interact with BeatPass. This guide covers all configuration options available in your account.

---

## Accessing Settings

Navigate to settings:

1. Click your avatar in the top right
2. Select **Settings** from the dropdown
3. Or go directly to: `open.beatpass.ca/settings`

---

## Profile Settings

### Basic Information

| Setting | Description |
|---------|-------------|
| Display Name | Your public username |
| Bio | Description shown on your profile |
| Profile Photo | Your avatar image |
| Header Image | Banner image on your profile |
| Location | Optional, shown publicly |

### How to Update

<Steps>
  <Step title="Go to Settings">
    Navigate to Settings > Profile.
  </Step>
  
  <Step title="Edit Fields">
    Click on any field to edit. Changes auto-save or require a Save button.
  </Step>
  
  <Step title="Upload Images">
    Click on photo areas to upload new images.
    - Profile: 400x400px minimum, square
    - Header: 1500x500px recommended
  </Step>
</Steps>

## Account Information

### Email Settings

| Setting | Description |
|---------|-------------|
| Email Address | Your login email (can be changed) |
| Email Verification | Verify to unlock all features |

<Note>
  Changing your email requires verification of the new address.
</Note>

### Password

To change your password:

1. Go to Settings > Security
2. Click **Change Password**
3. Enter current password
4. Enter and confirm new password
5. Save changes

### Connected Accounts

Link social accounts for easier login:

| Provider | Features |
|----------|----------|
| Google | Quick login, profile import |
| Facebook | Quick login |
| Apple | Quick login |

Manage at Settings > Accounts > Connected Accounts.

## Notification Settings

### Email Notifications

| Category | Options |
|----------|---------|
| **Activity** | New followers, likes, comments |
| **Messages** | New messages, beat request responses |
| **Earnings** | Payout notifications, earnings updates |
| **Marketing** | Newsletter, promotions, features |

### Push Notifications

Enable/disable browser notifications for:
- Real-time messages
- Important alerts
- Achievement unlocks

### Configuring Notifications

1. Go to Settings > Notifications
2. Toggle categories on/off
3. Set frequency preferences (immediate, daily digest, weekly)
4. Save changes

## Security Settings

### Two-Factor Authentication

Enable 2FA for additional security using an authenticator app (TOTP):

1. Go to Settings > Security
2. Click **Enable 2FA**
3. Scan QR code with your authenticator app
4. Enter the 6-digit verification code
5. Save your recovery codes securely

<Note>
  BeatPass uses TOTP-based 2FA only. SMS authentication is not available.
</Note>

### Active Sessions

View and manage logged-in devices:

1. Go to Settings > Security > Sessions
2. Review all active sessions
3. Click **Revoke** on any unrecognized sessions

### Login History

View recent login activity:
- Date and time
- Device/browser
- Location (approximate)

## Danger Zone

### Account Deletion

<Warning>
  Account deletion is permanent and cannot be undone.
</Warning>

To delete your account:

1. Go to Settings > Account
2. Scroll to **Danger Zone**
3. Click **Delete Account**
4. Confirm by entering your password
5. Confirm final deletion

This removes:
- All profile information
- Uploaded tracks (if producer)
- Library and playlists
- Transaction history
- Messages
