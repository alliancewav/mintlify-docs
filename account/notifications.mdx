## Overview

BeatPass sends notifications to keep you informed about activity, earnings, and platform updates. Customize your preferences to get the notifications that matter to you.

---

## Notification Types

### Activity Notifications

| Notification | Description |
|--------------|-------------|
| New Follower | Someone followed your profile |
| Track Like | Someone liked your track |
| Track Save | Someone saved your track to library |
| Comment | Someone commented on your track |
| Mention | Someone mentioned you |

### Message Notifications

| Notification | Description |
|--------------|-------------|
| New Message | Direct message received |
| Beat Request Response | Producer responded to your request |
| Collaboration Invite | Someone invited you to collaborate |

### Earnings Notifications (Producers)

| Notification | Description |
|--------------|-------------|
| Payout Processed | Monthly payout sent |
| License Purchased | Someone bought a license |
| Earnings Milestone | Hit earnings threshold |
| Contribution Update | Monthly contribution summary |

### System Notifications

| Notification | Description |
|--------------|-------------|
| Security Alert | Unusual login activity |
| Account Update | Profile or settings changed |
| Platform Announcement | Important platform news |
| Feature Update | New features available |

## Delivery Channels

### Email

Best for:
- Important updates you shouldn't miss
- Earnings and payout information
- Weekly/monthly summaries
- Marketing and newsletters

### Push Notifications

Best for:
- Real-time alerts (messages, activity)
- Urgent security notifications
- Achievement unlocks

### In-App Notifications

Always available:
- Bell icon in navigation
- Shows unread count
- All notification types

## Configuring Preferences

### Accessing Settings

<Steps>
  <Step title="Open Settings">
    Click your avatar > Settings.
  </Step>
  
  <Step title="Select Notifications">
    Navigate to the Notifications section.
  </Step>
  
  <Step title="Customize">
    Toggle each notification type on/off for each channel.
  </Step>
  
  <Step title="Save">
    Changes save automatically.
  </Step>
</Steps>

## Frequency Options

### Email Digest Settings

Instead of individual emails, receive summaries:

| Option | Description |
|--------|-------------|
| Immediate | Each event triggers an email |
| Daily Digest | One email per day with all activity |
| Weekly Digest | One email per week summarizing activity |
| Off | No emails for this category |

### Quiet Hours

Set times when push notifications are silenced:

1. Go to Notifications > Quiet Hours
2. Set start and end times
3. Select days (weekdays, weekends, all)
4. Save preferences

<Note>
  Security and urgent notifications bypass quiet hours.
</Note>

## Managing Notifications

### Viewing Notifications

1. Click the bell icon in navigation
2. See all recent notifications
3. Click to view details
4. Mark as read/unread

### Clearing Notifications

- **Clear One**: Swipe or click X on individual notification
- **Clear All**: Click "Mark all as read" at top
- **Auto-clear**: Read notifications clear after 30 days

---

## Unsubscribing

### Email Unsubscribe

Every email includes an unsubscribe link:

1. Click "Unsubscribe" at bottom of email
2. Choose to unsubscribe from that category or all
3. Confirm choice

### Complete Opt-Out

To stop all marketing communications:

1. Go to Settings > Notifications
2. Scroll to Marketing section
3. Toggle all options off
4. Save changes

<Note>
  You'll still receive essential account and security emails regardless of preferences.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Not receiving emails">
    - Check spam/junk folder
    - Add notifications@beatpass.ca to contacts
    - Verify email is correct in settings
    - Check notification preferences are on
  </Accordion>
  
  <Accordion title="Push notifications not working">
    - Ensure browser notifications are allowed
    - Check device notification settings
    - Try re-enabling in settings
    - Clear browser cache
  </Accordion>
  
  <Accordion title="Too many notifications">
    - Switch to digest mode for activity
    - Disable low-priority notifications
    - Set quiet hours
  </Accordion>
  
  <Accordion title="Missing notifications">
    - Check in-app notification center
    - Verify category isn't disabled
    - Look in email spam folder
  </Accordion>
</AccordionGroup>

## Best Practices

### For Producers

Recommended settings:
- **On**: License purchases, payouts, beat request matches
- **Digest**: Likes, follows, comments
- **Off**: Marketing (unless you want updates)

### For Artists

Recommended settings:
- **On**: Messages, beat request responses
- **Digest**: New releases from followed producers
- **Off**: Activity from others (unless curating)
